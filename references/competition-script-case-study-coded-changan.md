## 二、选题策略

**选择"漫影新叙事"方向的理由：**

1. **规则红利**：该方向明确写了"适配AI制作"——而短剧大师的AI生图生视频能力正是降维打击
2. **差异化**：其他方向（现实题材、都市情感）竞争激烈，漫影新叙事是蓝海
3. **元叙事**：用AI讲一个关于AI考古的故事——这个故事本身就适合用新媒介呈现

**选题决策流程：**
```
大赛四大方向 → 逐一匹配度分析 → 选最有规则红利的 → 提3套概念方案让用户选 → 用户定方向 → 开写
```

## 三、创作流程实录

### 3.1 从一句话到完整剧本

```
Step 1: 一句话介绍（4版候选→用户选A版）    ─── 5min
Step 2: 故事大纲（500+字）                  ─── 15min
Step 3: 人物小传（含戏剧功能标注）          ─── 15min
Step 4: 第1集剧本                           ─── 20min
Step 5: 打磨（5项改进）                    ─── 15min
Step 6: 第2-3集                             ─── 25min
Step 7: 第4-8集·终章                       ─── 40min
Step 8: 后记+创作说明                      ─── 10min
```

### 3.2 打磨环节（5项标准改进）

每次出初稿后必须执行：

| # | 检查点 | 本案例实操 |
|---|--------|-----------|
| 1 | 一句话介绍是否有"钩子"？ | 原版长句无节奏 → 改为三层递进+破折号停顿→落在一个"人"上 |
| 2 | 科幻/奇幻设定是否有技术解释？ | "AI自动启动"缺解释 → 增加"递归坐标系"技术铺垫 |
| 3 | 人物小传是否有戏剧功能标注？ | 补：求知者/引路人/盟友/对抗力量/背景锚点 |
| 4 | 团队简历是否完整？ | 从填表大师skill提取全量履历（教育/职务/专利/软著/标准/项目） |
| 5 | 缺原创声明/参赛授权书？ | 生成完整法律文本 |

### 3.3 人物塑造技法

**戏剧功能标注法**（评审看剧本第一眼就要看到这个）：

| 角色 | 戏剧功能 | 角色弧线 |
|------|---------|---------|
| 林小禾 | 求知者/解码者——观众的眼睛，负责推开那扇门 | "只想偷懒"→跨越千年的"翻译者" |
| 十七 | 引路人/谜面本身——既是钥匙也是答案 | 守望者→看见千年后的长安→自我和解 |
| 赵启明 | 盟友/情感锚点——连接科技线和人情线 | 半信半疑→无条件支持→暗恋表白（直男式） |
| 何砚秋 | 对抗力量/影子——"守护vs掠夺"的镜像对立 | 掠夺者→自我觉醒→"我想做我自己" |
| 吴道子 | 背景锚点/情感支线——解锁下一层壁画的钥匙 | 暴躁老头→"走吧，别回头"（全剧情感爆发点） |

## 六、经验总结

1. **选题优先选"有规则红利"的方向**——漫影新叙事明确写了"适配AI制作"，等于给短剧大师开了绿灯
2. **一句话介绍要三层递进**：设定→奇观→情感（本案例：AI解码→藏着一座城→等了一千二百年的人）
3. **科幻设定必须有技术解释**：不能"AI莫名自动启动"，必须铺垫"递归坐标系""语义向量重建"
4. **人物小传必须有戏剧功能+角色弧线**：这是编剧评审的第一眼筛选标准
5. **团队简历必须从填表大师全量提取**：学历/职务/专利号/软著号/标准号一个不能少
6. **反派不是工具人**：何砚秋从"掠夺者"走到"自我觉醒"——反派有弧线，故事才有重量
