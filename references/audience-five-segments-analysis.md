|------|------|

| 平台 | 抖音漫剧、红果漫剧、B站 |
| 市场规模 | 漫剧TOP2题材（逆袭、玄幻仙侠断崖领先） |

## 四、全受众覆盖的用户迁移路径

基于《发配边关·罪妻开荒养出战神》22亿播放的反推：

```
入口层（漫剧吸男性）
  ┌─────────────────────────┐
  │ 末世背景 → 三幻男（18-30）│
  │ 异能+战神 → 男频（24-40） │
  └─────────┬───────────────┘
            ↓
留存层（种田留女性）
  ┌─────────────────────────┐
  │ 开荒产粮 → 女频（25-45） │
  │ 大女主逆袭 → 大女主（20-35）│
  └─────────┬───────────────┘
            ↓
扩散层（温情扩全龄）
  ┌─────────────────────────┐
  │ 过日子不撕 → 温情（25-50）│
  │ 合家欢 → 银发族（50+）   │
  └─────────────────────────┘
```

**关键数字：**
- 漫剧用户24-40岁占比 **55%+** → 从三幻/末世切入
- 女性偏好度TGI **159** → 女性被种田/大女主吸引后粘性极高
- 漫剧与短剧付费重合仅 **38%** → 62%为新增付费群体

## 五、对短剧大师项目的启示

1. **漫剧是破圈的起点**（吸男频+年轻人）+ **种田温情是留存核心**（留女频+全年龄）
2. 「末世×穿越×种田×大女主」的融合恰好架起了从漫剧到短剧的桥
3. **温情不搞冲突**是目前市场最大的稀缺品——甜宠/逆袭都已红海，温情种田还在蓝海窗口
4. 漫剧女频内容严重稀缺（2025年TOP100仅6部），是6-12个月的先发窗口
