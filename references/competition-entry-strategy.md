## 一、参赛决策五步法

### Step 1: 读透规则

从赛事通知中提取以下7个关键字段，填入速查表：

| 字段 | 说明 |
|------|------|
| 主办方 | 官方/半官方/商业平台 |
| 截止日期 | 绝对红线 |
| 申报类别 | 剧本类/成片类/人才类 |
| 主题/赛道 | 对号入座的关键 |
| 素材要求 | 字数、集数、格式 |
| 权益/奖金 | 现金/流量/平台对接/政策扶持 |
| 申报方式 | 官网/邮箱/邮寄 |

### Step 2: 匹配优势

评估短剧大师的武器库与赛事要求的契合度：

| 短剧大师能力 | 匹配赛道类型 |
|-------------|-------------|
| AI漫剧 / 3D画风 | 科技类、AIGC类赛道 |
| 温情种田·不搞冲突 | 文旅、非遗、现实题材类赛道 |
| 22大师军团·全流程 | 剧本类（好故事）赛道——最强优势 |
| 22亿爆款方法论 | 任何允许说明创作实力的赛道 |
| 北京创业科创科技中心 | 北京本地赛事（含区域保护政策） |

### Step 3: 选赛道

核心原则：**不要硬塞，要自然契合**

检查清单：
- 作品立意是否自然贴合赛道主题？
- 赛道是否有明确的技术导向（AIGC鼓励）？
- 赛道是否有你的差异化优势（其他参赛者做不了你的风格）？
- 是否需要副赛道？副赛道是加分不是减分

### Step 4: 定制内容

根据赛道要求全新创作，不要套旧作：

| 类别 | 最低要求 | 短剧大师优势 |
|:----:|---------|-------------|
| 好故事 | 大纲≥5000字 + 剧本≥1/2集 | 321部剧本库、剧本大师一键出稿 |
| 好作品 | 样片≥1/3集 + 片花 | 小云雀API+pippit引擎快速出片 |

### Step 5: 同步申报

同一作品可同时报的多个渠道（注意不冲突）：

| 渠道类型 | 可并行申报？ |
|---------|:-----------:|
| 主办方自有赛事 | ✅ 同一作品一般不冲突 |
| 多个区域赛 | ❌ 创客中国体系内只能报1个 |
| 项目参赛 + 机构入库 | ✅ 完全不冲突 |
| 不同赛事体系 | ✅ 一般可并行（如追光+创客北京） |
