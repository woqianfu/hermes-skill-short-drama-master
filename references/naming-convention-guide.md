## 一、市场规律

### 高频钩子词（来自1000部投流短剧分析）

| 排名 | 词 | 频率 | 意义 |
|------|-----|------|------|
| 1 | 总裁 | 47/1000 | 身份锚点 |
| 2 | 夫人 | — | 女性向身份 |
| 3 | 离婚 | 动词第1 | 冲突钩子 |
| 4 | 绝世/战神/修罗 | — | 男频强力词 |
| 5 | 开局/穿越/重生 | — | 时空钩子 |

### 命名底层公式

```
身份反差 + 情绪强词 + 悬念
```

- **身份反差**：罪妻/罪妃/罪奴 vs 化学/程序员/兽医（古代身份×现代技能）
- **情绪强词**：炸了/藏不住/震惊/逆袭（让人想点开）
- **悬念**：她是个/开局/从零开始（让人想知道怎么回事）

## 二、边关罪妻系列命名铁律

| 原则 | 要 | 不要 |
|------|-----|------|
| 前缀借势 | "发配边疆"借"发配边关"22亿IP | 不要另起炉灶 |
| 视觉冲击 | "把边关炸了"——画面感+双关钩子 | 不要双关梗（"制皂"太绕） |
| 情绪直接 | 用暴力动词（炸、干、搞） | 不要正经叙事（"改变边疆"太正） |
| 低理解门槛 | 副标题5秒内看懂 | 不要依赖背景知识（"九年义务教育"需理解设定） |

### 对比例

| ❌ 被否 | ✅ 通过 | 原因 |
|---------|---------|------|
| 罪妻制皂：我在边疆搞化学 | 发配边疆：罪妻把边关炸了 | "制皂"双关太绕；"炸了"视觉冲击 |
| 边关罪妻：化学改变边疆 | 同上 | 太正经，没有情绪 |
| 边关罪妻：九年义务教育够用了 | 同上 | 需理解穿越设定，门槛高 |

## 三、漫剧命名速查

| 要素 | 推荐写法 |
|------|---------|
| 格式 | `前缀：身份+动词/副词` |
| 前缀 | 发配边疆 / 穿越成罪妻 / 开局 |
| 身份 | 罪妻 / 罪妃 / 罪奴 |
| 技能 | 化学老师 / 程序员 / 兽医 |
| 动词 | 炸了 / 搞 / 藏不住 / 逆袭 |
| 全名长度 | 7-10字最佳，≤12字 |
