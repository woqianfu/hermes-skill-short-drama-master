| 提示词 | 摄像机从低处平稳上升，画面从地面细节逐渐扩展到高处俯瞰，展示空间的高度变化，或反之从高处下降到低处。 |

|------|------|
| 景别 | 中景，主体始终居中 |

| 提示词 | 摄像机围绕主体做平滑的弧形运动。从正面开始，移动到左侧，再到背面，然后到右侧，最后回到正面。主体始终居中，背景动态旋转，营造三维空间感。 |

### ⑨ 变焦（光学变焦）
| 维度 | 内容 |
|------|------|
| 景别 | 中景到眼部特写 |
| 运镜动作 | 仅通过改变镜头焦距使画面放大或缩小，摄像机不动 |
| 速度/方向 | 快速 |
| 主体动作 | 主角的眼睛突然放大，瞳孔中倒映出爆炸火光 |
| 氛围/光影 | 背景压缩感明显，惊悚感强烈 |
| 提示词 | 摄像机位置不动，仅通过改变镜头焦距使画面放大或缩小。变焦推进时，背景压缩感明显，主体显得扁平；变焦拉远时，画面视野变宽。 |

### ⑩ 跟焦（焦点转移）
| 维度 | 内容 |
|------|------|
| 景别 | 双人近景，景深浅 |
| 运镜动作 | 改变焦点位置，从前景模糊的后脑勺平滑转移到背景清晰的脸 |
| 速度/方向 | 平滑 |
| 主体动作 | 对话者嘴角露出冷笑，前景人物静默 |
| 氛围/光影 | 背景虚化，引导观众注意力 |
| 提示词 | 摄像机固定，改变焦点位置。开始时前景人物模糊、背景清晰，然后焦点平滑转移到前景人物，使其变得清晰，背景同时虚化。常用于引导观众注意力。 |

### ⑪ 手持晃动

|------|------|
| 景别 | 特写 |

| 速度/方向 | 无规则抖动，轻微水平摇晃和上下颠簸 |
| 主体动作 | 主角蜷缩在墙角急促呼吸，身体随镜头晃动 |
| 氛围/光影 | 灯光无规则扫过，战地纪实感，粗粝真实 |
| 提示词 | 画面模拟手持摄像机拍摄，存在持续的、细微的抖动和不规则运动——轻微的水平摇晃、偶尔的上下颠簸以及松散的构图。这种效果创造出粗粝、纪实的真实感，常用于恐怖片或纪录片。 |

| 提示词 | 摄像机向后拉的同时镜头向前变焦，两者同步进行。主体在画面中保持大小不变，但背景看起来被拉伸和扭曲，产生强烈的眩晕和心理不安感。 |

### ⑬ 航拍/无人机

| 速度/方向 | 急速下坠再上升 |
| 主体动作 | 无人机跟拍一艘飞行摩托掠过水面，摩托加速逃离 |
| 氛围/光影 | 水雾弥漫，运动流畅不受空间限制，展示宏大场景 |
| 提示词 | 从高空鸟瞰视角开始，无人机快速向前俯冲，贴着水面或树梢上方掠过，然后平滑拉起并再次升高。整个运动流畅且不受空间限制，适合展示宏大场景。 |

| 速度/方向 | 自然晃动 |
| 主体动作 | 主角低头看见机械手臂，抬头时敌人冲出，HUD显示目标锁定 |
| 氛围/光影 | 强烈的沉浸感，画面边缘有呼吸起伏 |
| 提示词 | 镜头精确呈现角色自己眼睛所看到的画面。摄像机随着头部运动而运动——快速摇动、行走时的轻微上下起伏，以及偶尔的眨眼或手部进入画面。它创造了强烈的沉浸感。 |

### ⑮ 快速变焦（滑动变焦）

| 速度/方向 | 瞬间完成 |

| 氛围/光影 | 喜剧式惊吓，强调惊讶或顿悟 |
| 提示词 | 非常快速且突兀的变焦推近，通常在不到一秒内完成。画面瞬间从较远的景别猛冲至特写，用于强调惊讶、顿悟或喜剧节拍。 |

| 提示词 | 摄像机置于一个角色的肩膀后方，显示其后脑勺。然后随着对话的自然节奏摇向另一个角色。这种镜头营造出偷听或客观观察的感觉，常用于双人对话。 |

### ⑰ 横移跟随
| 维度 | 内容 |
|------|------|
| 景别 | 侧面跟拍，主体居中 |
| 运镜动作 | 摄像机横向移动，速度与奔跑主体完全相同，背景横向模糊 |
| 速度/方向 | 与主角冲刺速度一致 |
| 主体动作 | 主角在跑步机上冲刺，汗水飞溅，身体前倾 |
| 氛围/光影 | 背景时速表旋转加快，平滑而充满能量的追逐感 |
| 提示词 | 摄像机横向移动，速度与奔跑的主体完全相同。主体始终居中，背景横向模糊。这创造出平滑、充满能量的追逐感，常用于奔跑或赛车场景。 |

### ⑱ 主观晃动
| 维度 | 内容 |
|------|------|
| 景别 | 第一人称主观视角 |
| 运镜动作 | 摄像机不规律移动，结合旋转、垂直摇摆和突然的方向改变 |
| 速度/方向 | 醉酒般颠簸，剧烈无规则 |
| 主体动作 | 主角倒地，视线模糊重影，努力摇头试图聚焦 |
| 氛围/光影 | 画面边缘有眩晕光晕，然后逐渐聚焦，表现醉酒/眩晕/受伤状态 |
| 提示词 | 摄像机不规律移动，模仿角色不稳定的身体状况——醉酒、眩晕或受伤。画面结合了旋转、垂直摇摆和突然的方向改变，通常配合模糊效果，表现角色生理状态。 |

### ⑲ 越轴/跳跃运镜
| 维度 | 内容 |
|------|------|
| 景别 | 对话双人中景 |
| 运镜动作 | 摄像机位置突然跨越180°轴线跳跃，非连续运动 |
| 速度/方向 | 突然切换 |
| 主体动作 | 角色位置互换，再跳回原角度，产生空间错乱 |
| 氛围/光影 | 节奏失调，迷失感，风格化的快节奏效果 |
| 提示词 | 摄像机位置并非连续运动，而是突然跨越一百八十度轴线跳跃。角色面对的方向看似翻转，制造出有意的迷失感或风格化的快节奏效果。此手法在AI中常需后期剪辑实现。 |

### ⑳ 长镜头（一镜到底）
| 维度 | 内容 |
|------|------|
| 景别 | 后背跟随，连续变化景别 |
| 运镜动作 | 单一不间断的镜头，组合推、拉、摇、跟等多种运动，无剪切 |
| 速度/方向 | 手持跟拍，快慢结合，根据动作节奏调整 |
| 主体动作 | 主角翻窗→滑下楼梯→跃过柜台→钻进电梯，一气呵成 |
| 氛围/光影 | 无剪辑痕迹，沉浸式连续叙事，高难度动作场面 |
| 提示词 | 一个单一的、不间断的镜头，组合多种摄像机运动（推、拉、摇、跟等）来讲述一个连续的场景，没有任何剪切。它要求精确的调度，常用于沉浸式叙事或高难度动作场面。 |

## 短剧场景运镜选择

| 场景类型 | 推荐运镜 | 理由 |
|------|------|------|
| 日常对话 | 拉镜头 / 跟焦 / 过肩摇 | 自然不打断叙事 |
| 情绪转折 | 推镜头 / 变焦 / 快速变焦 | 强调情感变化 |
| 武打/追逐 | 移镜头 / 跟镜头 / 横移跟随 | 速度感和纵深感 |
| 觉醒/名场面 | 俯仰镜头 / 环绕镜头 / 升降镜头 | 雄伟、宏大 |
| 悬念/恐怖 | 手持晃动 / 希区柯克 / 主观晃动 | 不安感、心理暗示 |
| 环境交代 | 摇镜头 / 航拍 | 展示空间全貌 |

## 速度术语标准化

| 速度 | 使用场景 | 英文关键词 |
|------|------|------|
| 缓慢 | 叙事、情绪、对话 | slow, gentle, subtle |
| 匀速 | 拉、移、跟等基础运镜 | steady, even, constant |
| 快速 | 变焦、追逐、突发 | fast, rapid, abrupt |
| 瞬间 | 快速变焦、跳跃 | instant, snap, less than 1s |
| 不规则 | 手持、主观晃动 | irregular, shaky, unstable |

## 光影/氛围关键词

| 氛围类型 | 关键词 | 适配运镜 |
|------|------|------|
| 情绪/心理 | emotional, tense, intimate, dramatic | 推镜、变焦、跟焦 |
| 环境/空间 | expansive, revealing, vast, grand | 摇镜、航拍、升降 |
| 动作/速度 | kinetic, energetic, dynamic, urgent | 移镜、跟镜、横移跟随 |
| 不安/恐怖 | uneasy, disorienting, vertigo, claustrophobic | 希区柯克、手持、主观晃动 |
| 纪实/沉浸 | raw, documentary, intimate, immersive | 手持、第一人称、长镜头 |
