# 即梦AI-视频生成3.0 Pro API 参考

> 来源: 火山引擎官方文档 https://www.volcengine.com/docs/85621/1777001 (2025.12)
> 短剧大师映射: Phase 3 制作交付 → API参数落地

## 接口信息

| 项目 | 值 |
|------|-----|
| 接口地址 | `https://visual.volcengineapi.com` |
| 请求方式 | POST |
| Content-Type | application/json |
| req_key | `jimeng_ti2v_v30_pro` |

## 核心参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|:--:|------|
| prompt | string | 文生视频必选 | ≤400字符(建议), ≤800字符(最大) |
| seed | int | 可选 | -1=随机, 正整数可复现 |
| frames | int | 可选 | 121(5s) / 241(10s), 默认121 |
| aspect_ratio | string | 可选 | 16:9/4:3/1:1/3:4/9:16/21:9, 默认16:9 |
| binary_data_base64 | array | 可选 | JPEG/PNG, ≤4.7MB, ≤4096px |
| image_urls | array | 可选 | 图片URL |

## 输出分辨率

| 21:9 | 16:9 | 4:3 | 1:1 | 3:4 | 9:16 |
|------|------|-----|-----|-----|------|
| 2176×928 | 1920×1088 | 1664×1248 | 1440×1440 | 1248×1664 | 1088×1920 |

## 错误码

| 码 | 含义 | 可重试 |
|:---:|------|:---:|
| 50412 | 文本审核不通过 | 否 |
| 50413 | 版权词拦截 | 否 |
| 50411 | 图片审核不通过 | 否 |
| 50516 | 视频后审核不通过 | 可 |
| 50429 | QPS超限 | 可 |
| 50430 | 并发超限 | 可 |
| 50500 | 内部错误 | 可 |

## 任务状态

`in_queue` → `generating` → `done` / `not_found` / `expired`
