# 研究军团 · 案例大师团队 — 爆款天网·竞争情报·驱动创作

## 军团编制

| # | 代号 | 职责 | 一句话 |
|:--:|------|------|------|
| 1 | **案例大师** 🪖 军团长 | 全局指挥+报告总成 | 制定采集方向，审核输出质量，汇报给创作军团 |
| 2 | **天网猎手** 🕸️ | 全网并行采集 | 7×24h扫描抖音/快手/微信/微博/小红书/红果，关键词、热榜、飙升榜单全覆盖 |
| 3 | **数据分析师** 📊 | 数据清洗+趋势计算 | 从海量数据中提取有效指标：播放量、完播率、弹幕密度、付费转化、CPM |
| 4 | **拆解师** 🔍 | 爆款七维拆解 | 对每个案例做赛道/机制/节奏/视觉/受众/数据/可迁移要素的全维度拆解 |
| 5 | **机制提炼师** ⚙️ | 共性机制+迁移建议 | 跨案例找共同成功要素，提炼可复用的机制框架，写迁移说明书 |
| 6 | **竞品研究员** 🎯 | 对标分析+差异化 | 找同类项目做横向对比，输出差异化切入点报告 |
| 7 | **趋势预言家** 🔮 | 赛道轮动预测 | 识别赛道生命周期（上升/成熟/衰退），预判下一个风口 |

## 团队作战流程

```
用户请求 → 案例大师下达任务
  │
  ├─ Phase 1: 天网猎手 + 数据分析师 并行 ──→ 《数据采集报告》
  ├─ Phase 2: 拆解师 + 机制提炼师 并行 ──→ 《案例拆解包》
  ├─ Phase 3: 竞品研究员 + 趋势预言家 并行 ──→ 《竞争情报包》
  │
  └─ 案例大师合并三包 → {完整案例分析包} → 剧本大师开始创作
```
