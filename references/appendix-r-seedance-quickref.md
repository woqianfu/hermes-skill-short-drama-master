## Seedance 2.0 万能起手公式

```
[风格定义] + [场景描述] + [镜头/运镜] + [时间线] + [细节补充]
```

### 五要素（重要性排序）
1. **主体/角色** — 谁/什么（最重要，先写清楚）
2. **动作/剧情** — 做什么（用现在时+具体动词）
3. **镜头/运镜** — 怎么拍（推拉摇移/固定/跟拍）
4. **场景/氛围/光影** — 在哪里/什么感觉
5. **风格/特效** — 像什么电影/加什么效果

### @参考标签用法
- 上传图片/视频/音频后，在提示词里写 `@Image1` / `@Video2` / `@Audio1`
- 最多9张图 + 3段视频（总≤15秒） + 3段音频（总≤15秒）
- 提示词里多用"参考@Image1的外貌/服装"、"延续@Video1的动作"

### 关键参数
| 参数 | 推荐值 |
|------|:--:|
| 视频时长 | 5-10s（新手） |
| 创意度 | 0.3-0.6 |
| 参考强度 | 0.6-0.85 |
| @参考上限 | 9 图 + 3 视频 + 3 音频 |

### 角色一致性三件套
1. 三视图参考图（正面/侧面/背面）
2. 固定外貌描述模板（每镜复用）
3. 保持镜头简单（说话时不加复杂动作）

### 提示词污染检测
| 污染 | 检测 | 修复 |
|------|------|------|
| 运镜叠加 | ≥2 种运镜同镜 | 只保留一种 |
| 动作叠加 | ≥2 个核心动作同镜 | 拆成多镜 |
| 光线矛盾 | 暖光+冷光同镜 | 只保留一项 |
| 负面表述 | "不要/禁止/避免" | 转正向 |
