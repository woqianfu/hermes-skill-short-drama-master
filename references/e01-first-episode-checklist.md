## 一、爽点检测（🔴 首集专有·不可跳过）

- [ ] 观众是否在前 **15秒** 内获得一个「我比剧中人知道的多」的时刻？
- [ ] 这个时刻是否通过**局外视角**（NPC/倒影/影子/镜子）呈现，而非主角主动炫技？
- [ ] 是否至少有一个剧中人"看到但否定"了这个真相？（看到的 NPC 自己说服自己"眼花了"）

> 通过条件：三项全✅。任一项❌ → 须注入爽点（见 §1.3.1a 三种方案）。
> **实战首选方案B**（理由：不破人设 + 短剧信息差=爽感核心 + 小云雀生产友好）。

### 方案B 实战模板（已在《苍生》E01 验证通过）

```
【镜2】· 中景 · 3s（插在镜1大远景和原镜2打水之间）
一个路人 NPC 无意中看到主角——通过水面/镜子/影子的倒影——
看到了另一个版本的主角（更年轻/更强大/更真实的形态）。
NPC 揉眼、怀疑自己、摇头走人。

模板要素：
- NPC：一个与主线无关的普通人（卖柴的/送菜的/路过的）
- 媒介：水面/镜子/影子/玻璃反光（不需特效）
- 倒影：展示主角的"真实形态"（青年身形/直脊背/黑发）
- NPC 反应：看到→震惊→自我否定（"眼花了"）
- 时长：3 秒（0-1s 看到 → 1-2s 震撼 → 2-3s 否定）
```

> 完整实战内容（含竖屏七法/光影/材质/运镜/暗线标注全字段）见下方第六节。

## 二、E01 专项修复清单（7项·已验证）

| # | 检查项 | 常见问题 | 修复 |
|---|--------|---------|------|
| 1 | **暗线D代价描述** | 三镜三种说法→自相矛盾 | 建「症状递进线」：镜1=隐匿→镜2=首个微兆→镜3=代偿掩盖。每镜代价描述不重复、不矛盾、有递进 |
| 2 | **过渡方式选型** | 场景跳跃用"渐变" | 空间跳跃→硬切。渐变仅用于时间连续或情绪递进 |
| 3 | **光影基调控级** | 首集第二镜就用④神圣觉醒 | 建立集日常镜用②温暖日常。④仅用于灵脉初现/觉醒/战斗 |
| 4 | **运镜关键词** | 只有过渡方式，缺少主动运镜 | 每镜声明一种运镜（推/拉/摇/跟/固定），见 J.3.2 |
| 5 | **灵脉行为逻辑** | 三个动词无统一框架 | 建立「灵脉回应五级」：⑤强烈·共鸣 / ④温和·靠拢 / ③波动·倏聚散 / ②微弱·闪烁 / ①静默 |
| 6 | **角色行踪过渡** | 面馆门口→井边，中间动机缺失 | 加过渡段交代等待/跟踪/观察的时间 |
| 7 | **尾钩独立度** | 尾钩被嵌入末镜内部 | 独立为"═══ 尾钩 ═══"段落 + 标注精确秒数 |

## 三、暗线D代价三阶段递进模板（可直接复用）

```
镜1：代价100% / 症状隐匿——{年数}年的日常动作已成为肌肉记忆，代价在基线状态完全不可见
镜2：代价100% / {具体症状}（不可控的{延迟/颤抖/停顿}）——代价的首个微兆。肌肉记忆自动补偿
镜3：代价100% / 动作看似流畅——但仔细观察可见{隐藏特征}。不是不{异常}，是{异常}被藏进了动作的韵律里
```

## 四、灵脉回应五级框架（可直接复用）

```
⑤强烈·共鸣：金色微粒主动聚拢——灵脉对{角色A}的日常回应，{年数}年的习惯性共鸣
④温和·靠拢：金色微粒缓缓靠向{角色B}指尖——灵脉在试探外来修士的身份
③波动·倏聚散：金色微粒倏然聚拢又散开——如水面被投入石子，微弱但明确
②微弱·闪烁：金色微粒亮度瞬间升高后恢复——不仔细看不会发现
①静默·无反应：金色微粒无变化——灵脉未对{普通角色/物品}产生回应
```

## 六、方案B 实战完整示例（《苍生》E01 新镜2·可直接参考）

```
第01集 / 场01b / 石桥古井·晨 / 延炎 · 卖柴镇民（路过）

【镜2】· 中景 · 3s
【竖屏七法·②纵深分层】
【运镜·固定镜头——井口侧低位机位，水面倒影构图】
【光影基调·②温暖日常】过渡方式：硬切
【材质·前景】井沿青石水渍反光，水面微微荡漾
【材质·中景】延炎蹲在井边的倒影——水面以下三寸处，倒影的轮廓开始变形；
  水面上是他佝偻的肩膀，水面下三寸的倒影里，那个肩膀是平的、宽的、如三十岁的壮年
【材质·远景】石桥拱在晨雾中淡出，远处一个挑柴人的剪影停在桥上
【尺度锚点】井口直径约80cm / 水面距井沿约一米 / 倒影变形的临界线在水面下约三寸

内部节拍：
  0-1s：石桥另一端，一个卖柴的镇民挑着柴走过——他无意中朝井边望了一眼
        【爆点】信息揭露
  1-2s：镇民的目光扫过水面——他停住了。不是停步，是目光停住了。
        因为他看到水面上的老头和水面下的倒影——不是同一个人
        【爆点】视觉特效
  2-3s：镇民揉了揉眼睛。再看时——水面恢复如常。他摇了摇头，挑起柴继续走

【暗线A】灵脉状态：充沛——井水在倒影变形的三寸深度处，金色纹路密度突然升高
（灵脉在反射延炎的「真实形态」）
【暗线B】封印完整度：7/7
【暗线C】遗迹威胁：沉睡
【暗线D】延炎代价：100% / 延炎本人未察觉——井水倒影是他三百年前的「灵体残影」，非他主动释放
```
