# 穿越边疆×偏门职业 创作方法论

> 短剧大师™ 编剧工具箱
> 核心公式：穿越边疆 × 偏门职业 × 种田开荒 × 双强温情 × 轻喜剧
> 对标：德鲁·巴里摩尔原则 — 所有高度结构化的内容都可以被套用到预设的框架里，再进行定向填装

---

## 一、核心架构（不动）

```
身份：现代专业技能者 → 穿越成边疆罪奴/罪妻
处境：发配边疆，身无分文，从零开荒
成长：产品/技术升级线 + 财富积累线 + 情感线
基调：轻喜剧为主，温情为底，不搞冲突
```

## 二、变体公式

```
穿越边疆 × [偏门职业] × [产品升级链] × [与男主的关系]
```

## 三、开荒节奏模板（以化学老师为例）

### 第一阶段：活下来（第1-3集）
- 工具：0成本原料
- 产品：纯手工，零成本
- 财富：从0到能吃饱饭
- 关键动作：发现资源 → 动手做 → 被嘲笑 → 验证成功

### 第二阶段：第一桶金（第4-10集）
- 工具：简单铁锅+陶罐
- 产品：提档升级，市场空白
- 财富：从吃饱饭到有积蓄
- 关键动作：产品迭代 → 打开市场 → 遇到男主 → 资源置换

### 第三阶段：技术拐点（第11-18集）
- 工具：专用设备
- 产品：军需级，降维打击
- 财富：从积蓄到大单
- 关键动作：技术突破 → 军事验证 → 男主认可 → 建工坊

### 第四阶段：规模化（第19-26集）
- 工具：多条产线
- 产品：系列化+品牌化
- 财富：从大单到产业
- 关键动作：招人→建厂→扩市场→影响力溢出

### 第五阶段：巅峰（第27-30集）
- 工具：工业化
- 产品：碾压全行业
- 财富：边疆首富
- 关键动作：技术碾压 → 全行业认证 → 财富巅峰 → S1钩子

## 四、60集结构安全公式

```
S1（30集）：开荒创业
  5阶段，每阶段6集
  每个阶段的起承转合：发现→试错→突破→回报→新问题→升级

S2（30集）：更大棋局
  新地图/新冲突/新对手
  最终落脚点：知识传世
```

## 五、60秒单集公式（复用）

```
0-5s     黄金钩子：第一帧抓住观众
5-15s    情境定位：这是哪/谁在干嘛
15-35s   推进/冲突：核心剧情
35-50s   高潮点：本集最燃/最暖瞬间
50-60s   悬念收尾：拉下一集
```

## 六、编剧自检8问

1. 前3秒有没有抓人的画面或冲突？
2. 15秒内观众能知道这是哪、谁在干嘛？
3. 15-35秒的核心推进有没有至少1个起伏？
4. 35-50秒的高潮是不是本集最燃/最暖/最炸的瞬间？
5. 50-60秒的钩子是不是真的让人想看下一集？
6. 去掉所有台词，光看画面能看懂剧情吗？
7. 有没有任何"上集说到"或废话开头？
8. 这一集单独拿出来，不看上下文，好看吗？
