## 核心原理

**"干净帧"规则**：首帧 = 角色无表演变形的静态帧。用中间帧做锚点会把微表情/手势变化复制到其他镜头。

## 三步骤

### Step 1：生成主镜头（Master Shot）
AI唯一一次"创造世界"的机会。用J.3五段式prompt生成完整的场景主镜头。

```
prompt: 角色A(红裙) + 角色B(蓝卫衣) + 咖啡馆对话 + 中景 + 暖色调 + 自然光
```

### Step 2：锁定主关键帧（Master Key Frame）
提取主镜头的**第一帧**作为"世界锚点"。要求：
- ❌ 不能是中间帧（会把微表情/手势变形复制到其他镜头）
- ✅ 必须是干净的、无表演动作的初始状态

### Step 3：生成设置帧（Setup Frames）
喂给AI，指令格式：
```
冻结角色和场景，把相机移动到[新角度]
```

**关键限制**：
- ❌ 不可从"中景裁剪放大假装特写" → 透视关系错乱
- ✅ 每个镜头独立生成设置帧
- ✅ 结果：所有镜头角色长相/服装/场景光照完全一致（起点=同一帧）

## Seedance 2.0 落地

### 多模态引用方式
```
prompt: 【@图片1】角色A的锚点帧 → 角色A从门口走进来
```

### 逐台词节拍锚定
长对白场景需要在每个台词节拍处截取关键帧，为该节拍独立生成对应角度镜头：

```
台词A: "你来了" → 截取帧 → 生成角色A正面镜头
台词B: "我等你很久" → 截取帧 → 生成角色B反应镜头
台词C: "走吧" → 截取帧 → 生成双人全景
```
