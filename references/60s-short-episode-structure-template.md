# 60秒短集结构模板

> 短剧大师™ 研究军团·编剧工具箱
> 适配平台：抖音/红果/河马等短剧平台（60秒以下平台不收）
> 适用题材：末世×穿越×种田×大女主（及其他强钩子类型）

---

## 一、总则

**60秒 = 一个完整的叙事闭环 + 一个钩子拉下一集**

每一秒都是钱。60秒内必须完成：**建立情境 → 制造冲突/推进 → 释放情绪 → 埋钩子**

### 核心原则

| 原则 | 说明 |
|------|------|
| **三秒定生死** | 前3秒没抓住观众，这集就废了 |
| **信息密度>信息量** | 不是说更多话，是让每句话都有用意 |
| **情绪大于逻辑** | 60秒不可能讲完整逻辑，情绪到位观众自己能脑补 |
| **钩子不能假** | 不能为悬念而悬念，钩子必须推进主剧情 |

---

## 二、60秒逐秒分配表（标准模板）

```
00:00 ───────── 05 ───────── 15 ───────── 35 ───────── 50 ───────── 60
  ↑              ↑               ↑               ↑               ↑
 黄金钩子     情境定位       推进/冲突         高潮点          悬念收尾
 (3-5s)        (5-15s)        (15-35s)         (35-50s)        (50-60s)
```

### 逐段说明

| 段落 | 时长 | 占比 | 功能 | 节奏特征 |
|------|------|------|------|---------|
| **①黄金钩子** | 0-5s | 8% | 第一帧就抓住眼球 | 危机/反转/高能开头 |
| **②情境定位** | 5-15s | 17% | 告诉观众"这是哪/谁/在干嘛" | 快速交代，不废话 |
| **③推进/冲突** | 15-35s | 33% | 核心内容——打怪/种田/对峙/发现 | 情绪爬升 |
| **④高潮点** | 35-50s | 25% | 本集最重要的瞬间 | 情绪峰值 |
| **⑤悬念收尾** | 50-60s | 17% | 拉下一集 | 戛然而止 |

---

## 三、逐段详解 + 示例

### ① 黄金钩子（0:00-0:05）

**目标：让本来要划走的拇指停下来。**

不是"在上一集我们说到……"，而是直接扔一个**冲突/危机/高能画面**。

**常见钩子类型：**

| 类型 | 适用场景 | 台词/画面示例 |
|------|---------|--------------|
| **危机突降** | 末世/末日 | 丧尸破门而入 / 地裂塌陷 / 敌人杀到 |
| **身份反转** | 大女主/穿越 | "你不是被发配的罪妇，你…你是木系异能觉醒者？" |
| **发现惊喜** | 种田/开荒 | 挖到地下泉水 / 灵植一夜长成 / 发现隐藏宝藏 |
| **对话爆点** | 任何类型 | "三天之内，这片荒地我要让它长出粮食来。" |
| **敌我对峙** | 战神/逆袭 | 刀架在脖子上 / 箭在弦上 / 两军对峙 |

**【正确示例 - 末世种田】**
> 画面：丧尸群逼近寨墙，女主手握一把种子站在田埂上
> 台词（第3秒）：「你们吃人，我种地。看谁熬得过谁。」

**【错误示例】**
> 画面：风景空镜 + 旁白"上集说到主角在末世艰难求生……"
> — 观众3秒内就划走了

---

### ② 情境定位（0:05-0:15）

**目标：用10秒让观众知道"这是谁、在哪、要干嘛"。**

不要旁白交代，要用**动作+对话+画面**自然带出。

**信息三要素（每集最多带两样）：**

| 要素 | 末世种田示例 | 大女主示例 | 双强示例 |
|------|------------|----------|---------|
| **空间** | 寨墙外/荒地/山洞 | 边关衙门/农田/集市 | 战场/营地/地牢 |
| **状态** | 缺水/缺粮/丧尸围城 | 被轻视/暗中蓄力 | 搭档受伤/被包围 |
| **目标** | "今晚之前找到水源" | "三天后粮仓验收" | "救出俘虏才能谈和" |

**定位不是回忆**。不要出现："我是被发配到这里的罪妇，前世被陷害……"
而是用行动暗示。例：
> 女主割下一把野草，根上沾着黑土
> → 观众自己会理解：她在找可种植的土地（不需要台词说"我要种地"）

**【正确示例 - 10秒内交代三要素】**
> 0:05 外景·荒地·女主蹲着扒土（空间：末世）
> 0:08 女主皱眉：土里有毒，难怪这里寸草不生（状态：发现问题）
> 0:12 女主站起，看向远处的绿洲：那边——可能有戏（目标：找可耕地）

---

### ③ 推进/冲突（0:15-0:35）— 核心20秒

**目标：推进本集的核心事件。**

这是剧情的"肉"。20秒内要完成一个**小循环**：

```
遇到问题 → 尝试解决 → 遇到新问题/部分解决
```

**不同题材的推进节奏：**

| 题材 | 15-20s（上半段） | 20-35s（下半段） |
|------|-----------------|-----------------|
| **种田线** | 发现资源 / 开始开垦 | 遭遇阻力（野兽/敌人/天灾） |
| **战斗线** | 对峙/开打 | 反转/险胜/惜败 |
| **社交线** | 谈判/对话/对峙 | 身份揭示/立场变化 |
| **探索线** | 发现线索/进入新区域 | 意外遭遇/触发剧情 |

**【正确示例 - 种田推进】**
> 0:15 女主往绿洲方向走，地面突然震动
> 0:20 一只变异兽从灌木冲出，挡在去绿洲的路上
> 0:25 女主抬手，指尖泛绿光——木系异能发动，藤蔓从地面爆出缠住变异兽
> 0:30 变异兽挣扎嘶吼，撕断藤蔓
> 0:35 女主面色一变：二阶变异兽……得绕路

关键：**不要平铺直叙**——每5秒至少有一个小起伏。

---

### ④ 高潮点（0:35-0:50）

**目标：本集的情绪峰值。**

观众看了35秒，等的就是这个瞬间。高潮可以是：

| 类型 | 末世种田示例 | 情绪效果 |
|------|------------|---------|
| **成果展示** | 绿洲的水/灵植长出来了 | 爽/治愈 |
| **身份觉醒** | 木系异能升级，大范围操控植物 | 燃 |
| **危机升级** | 更大的威胁出现了（要变天了/敌军来了） | 紧张 |
| **关系突破** | 男主默默帮女主挡了一刀 | 暖 |
| **智慧获胜** | 用种植知识反杀敌人 | 解气 |

**高潮要"炸"——画面要有冲击力，台词要有一句话金句。**

**【正确示例】**
> 0:35 女主走到绿洲边，手指插入泥土
> 0:40 绿光从她指尖扩散，土地以可见速度变黑、松软
> 0:45 女主站起来，微笑：
> 「这方圆十里——以后就是我的田了。」

---

### ⑤ 悬念收尾（0:50-0:60）

**目标：让观众不得不点下一集。**

**钩子三大原则：**
1. **不能虚**（不要"请看下集"）
2. **不能剧透**（不要预先展示下集画面）
3. **必须推进**（钩子必须是剧情不可或缺的一部分）

**好钩子的五个型：**

| 型 | 做法 | 示例（末世代入） |
|----|------|----------------|
| **危机迫近型** | 下集威胁已在当前画面中出现 | 远处的地平线上，黄沙卷起——那不是风沙，是尸潮 |
| **信息炸弹型** | 一句话颠覆认知 | "你的木系异能不是种地的——你是能催生上古神树的唯一血脉" |
| **身份揭露型** | 有人有秘密 | 男主解开衣襟，胸前有一个军队烙印 |
| **倒计时型** | 给下集一个时间压力 | "三天后有寒潮，不建好粮仓我们全得死" |
| **悬念对话型** | 关键信息说一半 | 远处有人喊：大人，京城来人了，说——（画面切黑） |

**【正确示例】**
> 0:50 女主看着眼前肥沃的黑土，满意点头
> 0:52 远处突然一声长啸
> 0:55 女主脸色一变，抬头：这是…九阶变异兽的声音
> 0:58 画面切黑，字幕：**它冲着绿洲来了**
> 0:60 END

错误示例：女主说"明天再说吧"→ 画面黑 → 观众不看了

## 四、各种题材 × 60秒结构速查

### 模板A：种田日常（温情/治愈向）

```
0-5s    发现好东西（惊喜表情）
5-15s   确认这是什么/怎么用（快速科普）
15-35s  动手操作/种植/烹饪/建设（治愈过程）
35-50s  成果展示（大丰收/美食出锅/房间建好）
50-60s  远处有人来了/收到了信（悬念）
```

### 模板B：打怪/逆境（紧张动作向）

```
0-5s    危机局面（怪物扑来/敌人包围/天灾）
5-15s   主角身处险境（快速交代位置+状态）
15-35s  战斗/应对（动作场面，每5秒一个变招）
35-50s  取胜/关键发现（击杀/逃生/得到关键物品）
50-60s  更大的威胁出现（Boss现身/旧主回归）
```

### 模板C：社交/权谋（对话/智商向）

```
0-5s    冲突对话（剑拔弩张）
5-15s   人物关系交代（谁和谁在哪里对峙）
15-35s  话术博弈/计谋展开（你来我往）
35-50s  反杀/逆转（身份揭露/暗棋落子）
50-60s  背后还有棋（第三者视角/更大的阴谋浮出）
```

### 模板D：发现/探索（剧情推进向）

```
0-5s    线索/发现（看到什么东西）
5-15s   这是什么/在哪（环境+背景）
15-35s  探索过程（步步深入/触发机关/遇到阻碍）
35-50s  核心发现（找到宝物/秘密/人/真相）
50-60s  代价出现（拿到东西但触发了警报/惊醒了什么）
```

## 五、编剧自检清单

每集写完对自己问这8个问题：

| # | 问题 | 过/不过 |
|---|------|--------|
| 1 | **前3秒有没有抓人的画面或冲突？** | ☐ |
| 2 | **15秒内观众能知道这是哪、谁在干嘛？** | ☐ |
| 3 | **15-35秒的核心推进有没有至少1个起伏？** | ☐ |
| 4 | **35-50秒的高潮是不是本集最燃/最暖/最炸的瞬间？** | ☐ |
| 5 | **50-60秒的钩子是不是真的让人想看下一集？** | ☐ |
| 6 | **去掉所有台词，光看画面能看懂剧情吗？** | ☐ |
| 7 | **有没有任何"上集说到"或废话开头？** | ☐ |
| 8 | **这一集单独拿出来，不看上下文，好看吗？** | ☐ |

**3个以上不过 → 重写。**

## 六、全季结构参考

### S1（正常节奏，养观众）

| 集次 | 功能 | 推荐模板 |
|------|------|---------|
| 1-5集 | 入坑（立世界观+主角+危机） | D（发现）/ B（逆境） |
| 6-20集 | 发展（种田/升级/建基地） | A（种田）/ B（逆境）交替 |
| 21-50集 | 深化（人物关系/冲突升级） | B/C/D 混用 |
| 51-66集 | 收束（前一季高潮+留下季钩子） | C/D 为主 |

### S2（短集节奏，已入坑观众）

| 集次 | 功能 | 推荐模板 |
|------|------|---------|
| 1-10集 | 新危机+新形态 | D（发现）为主 |
| 11-35集 | 节奏加速、信息密度提升 | A/B 各半，偶有C |
| 36-52集 | 终极冲突+结局+下一季钩子 | B/C |
