# 短剧·漫剧 命名规律指南

> 基于《发配边疆：罪妻把边关炸了》3轮命名实战 + 1000部爆款短剧剧名词频分析
> 来源：数英1000部短剧剧名分析 + 本session实战验证

---

## 一、爆款剧名关键词排行

### 名词（出现频率从高到低）
总裁(47/1000)、夫人、离婚、战神、修罗、女帝、总裁夫人、保镖、神医、绝世

### 动词（出现频率从高到低）
离婚、天降、藏不住、退婚

### ⚠️ 2026年AI漫剧趋势：剧名更短、更直接
- 漫剧用户（18-30岁男性为主）偏好：3-7字，一眼知道是什么题材
- 《斩仙台下》《末日寒潮》《洪荒：代管截教》 — 短+场景+动作
- 副标题公式：`[主角]：[动作]+[目标]`

---

## 二、命名失败模式（本Session已验证）

| 模式 | 反面例子 | 为什么不行 |
|------|---------|-----------|
| 双关/谐音梗 | 制皂（=制造/肥皂） | 绕，竖屏3秒看不懂 |
| 依赖背景知识 | 九年义务教育够用了 | 需要理解设定才能get到 |
| 太正经/使命感 | 化学改变边疆 | 不像爽剧，像主旋律 |
| 太抽象 | 罪妻的元素周期表 | 元素周期表不是钩子 |

## 三、王炎（woqianfu）命名偏好

```
✅ 前缀借势IP → "发配边疆：罪妻"（跟《发配边关》22亿做关联）
✅ 副标题暴力直接 → "把边关炸了"（双关+视觉冲击+轻喜）
✅ 全名≤9字
✅ 字面就能看懂，不需要动脑子
✅ 有"动感" — 炸/搞/开局/藏不住
```

---

## 四、命名字段填空模板

```
[前缀]：[身份] + [动作/结果]

示例：
发配边疆：罪妻 + 把边关炸了
"""
前缀（IP借势）  —— 发配边疆 / 发配边关 / 穿越 / 重生
身份（1-2字）   —— 罪妻 / 罪妃 / 罪奴
动作（3-6字）   —— 把边关炸了 / 开局搞化学 / 从零开荒
"""
```

## 五、一句话检查清单

提交前自问：

1. 不看任何背景信息，字面能看懂这剧讲什么？ ☐
2. 3秒内能决定点不点进来？ ☐
3. 标题本身能不能当15秒切片文案用？ ☐
4. 有没有双关/谐音/需要解释的梗？ ☐（有就删）
5. 是不是太正经了像主旋律片？ ☐（是就改）
