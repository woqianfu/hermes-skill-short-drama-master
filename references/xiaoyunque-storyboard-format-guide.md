# 小云雀短剧Agent 2.0 故事板提交格式

> 短剧大师™ × 小云雀交付管线

## 核心认知

小云雀短剧Agent 2.0 **可以直接理解剧本原文**，不需要手动制作分镜表。

| 能力 | 说明 |
|------|------|
| 剧本自动解析 | 上传后自动解析结构，生成分镜脚本 |
| 角色自动管理 | 全剧本扫描，自动保持角色一致性 |
| 分镜自动生成 | 自动分配镜头、场景、运镜指令 |
| 多画风支持 | 2D/3D/仿真人，自动匹配 |
| 上限 | 10万字剧本 |

## 剧本格式要求

### 必须包含的要素

| 要素 | 写法 | 作用 |
|------|------|------|
| 片头角色设定 | `【角色设定】` 区块 | Agent建立角色资产库 |
| 场景标签 | `【场景N：地点·时间】` | Agent按场景分段 |
| 镜头描述 | 分行写：镜头+环境+人物+动作 | Agent逐行解析 |
| 内心独白 | `[内心独白·语气]` + 另起行 | Agent识别旁白轨道 |
| 对话/台词 | `人物：` 独立一行，冒号后台词 | Agent识别角色对话 |
| 转场标记 | `→ [叠化/切黑/划入]` | Agent生成转场指令 |
| 剧终标记 | `→ [切黑]` + 下集字幕 | Agent识别结束点 |

### 格式对照

| 编剧可读版 | 小云雀提交版 | 原因 |
|-----------|-------------|------|
| `**SCENE 1 · 0:00-0:08**` | `【场景1：地点·时间】` | Agent按场景标记分段 |
| 散文式画面描述 | 分行：镜头→环境→人物→动作 | Agent逐行解析各要素 |
| `OS：` 混在描述里 | `[内心独白·语气]` 独立标签 | Agent识别独立旁白轨道 |
| 无角色描述 | `【角色设定】` 片头区块 | Agent建立角色资产 |
| 无转场 | `→ [叠化/切黑/划入]` | Agent生成转场指令 |

### 完整示例结构

```
【剧名】发配边疆：罪妻把边关炸了
【风格】3D漫剧，轻喜剧
【画幅】9:16竖屏

【角色设定】
林晚：女，25岁，化学老师穿越成罪奴。囚服，手有烧伤。
沈昭：男，30+，镇北大将军，糙汉，脸上有刀疤。

【场景1：教室→囚车·白天】
[镜头：快速闪切]
画面亮：一只手拍在周期表上。
女声：今天讲酸碱中和——
画面剧烈晃动。
[内心独白·平静]
我叫林晚，市一中化学老师。
[镜头拉开：她趴在颠簸的囚车上，镣铐，囚服破烂]
[内心独白]
下一秒——被发配边疆。
→ [叠化转场]
```
