## A · 人物叙事构图（16种）

> 聚焦人物心理和关系，适合短剧日常/情感/悬疑场景。

| # | 名称 | 一句话描述 | 适配情节 |
|:--:|:----|:----------|:--------|
| A01 | 门轴偏置构图 | 人物靠门轴·留出走或留的心理空间 | 犹豫开门/密会/离开 |
| A02 | 回望断点构图 | 已走过中心·回首制造信息缺口 | 告别/察觉被跟踪/重逢 |
| A03 | 夹层观察构图 | 上下层栏杆分割·窥视感 | 监视/等候信号/潜伏 |
| A04 | 余光联结构图 | 不直接对视·通过镜像/侧脸构建关系 | 试探/暗斗/猜忌 |
| A05 | 临界跨步构图 | 一脚在内一脚在外·命运转折瞬间 | 逃离/入局/蜕变 |
| A06 | 光缝塑形构图 | 窄光束只勾勒部分轮廓·其余沉入黑暗 | 审讯/独白/秘密揭露 |
| A07 | 末端守望构图 | 走廊/站台尽头·被排除感 | 等待/边缘/排挤 |
| A08 | 斜面静置构图 | 倾斜环境中静止·以静对动表达抗挣 | 心理压迫/幸存/失衡 |
| A09 | 反向通道人像 | 逆行于人群·价值冲突 | 孤勇/社会压力/逆流 |
| A10 | 双层窗景构图 | 窗框+倒影叠加·身份漂移感 | 旅途记忆/卧底/离家 |
| A11 | 侧坐失衡构图 | 只占坐具边缘重心偏移·不安感 | 谈判前夕/分手/等待 |
| A12 | 近远对峙构图 | 前景半身+远景全身·空间对抗 | 审问/跟踪/身份确认 |
| A13 | 群体裂口构图 | 人群裂开·主角在末端 | 围观/抓捕/冠军通道 |
| A14 | 轮廓贴边构图 | 面部贴边入镜·大部分画框留给环境 | 偷听/旁观/目击 |
| A15 | 远近同框异步 | 同框不同时空·倒影+主体双重叙事 | 闪回/心理分裂/时间迷宫 |
| A16 | 暗场聚点构图 | 大面积暗场只保留一处亮光 | 深夜行动/末日/孤独决断 |

## B · 情绪氛围构图（16种）

> 专注于营造特定情绪和心理氛围，适合悬疑/心理/情感高潮场景。

| # | 名称 | 提示词 | 适配情节 |
|:--:|:----|:-------|:--------|
| B01 | **压迫边角构图** | 主体被挤压在画面角落，大面积建筑墙面或阴影占据视野，制造被环境吞没的压迫感与失衡心理 | 焦虑／审讯前夕／职场压迫 |
| B02 | **镜面分裂构图** | 利用镜子、玻璃、水面反射形成双重人物或空间切割，强调自我撕裂、秘密与心理冲突 | 人格挣扎／悬疑独白／身份错位 |
| B03 | **窄缝窥视构图** | 通过门缝、柜缝、帘隙等极窄视野观察主体，画面信息受限，增强偷窥感与未知威胁 | 监视／偷看／恐惧来临 |
| B04 | **遮半失衡构图** | 主体仅有半张脸或半个身体处于光线中，另一半没入黑暗或被遮挡，突出隐秘与道德灰度 | 反派亮相／秘密交易／内心拉扯 |
| B05 | **倒影主叙构图** | 主体本体弱化，主要通过地面积水、镜面桌面、橱窗倒影完成叙事，营造虚幻与疏离 | 回忆／都市孤独／梦境片段 |
| B06 | **色块对撞构图** | 利用强烈互补色背景或灯光区块切分人物，形成情绪冲突与视觉张力 | 争执／霓虹都市／情绪爆发 |
| B07 | **影子先行构图** | 画面主要呈现角色影子，本体弱化或出画，让影子承担叙事与威胁感 | 追凶／未知来者／危险逼近 |
| B08 | **玻璃阻隔构图** | 隔着起雾玻璃、雨窗、车窗拍摄主体，表面纹理干扰情绪表达，强化隔阂感 | 告别／错过／车站离散 |
| B09 | **环形包围构图** | 利用人群、灯带、拱门、桌席形成环状包围主体，突出被围观、审判或仪式感 | 会议施压／宗教仪式／群体凝视 |
| B10 | **贴地掠行构图** | 镜头极低贴近地面，前景地面纹理高速掠过，主体在中后景，形成临场冲击与潜行动势 | 潜行／逃亡／战术推进 |
| B11 | **失焦情绪构图** | 关键主体并非完全清晰，而是在浅景深中保留轻微失焦，配合光斑表达恍惚、醉意或回忆感 | 失恋／酒后／梦游状态 |
| B12 | **局部肢体构图** | 不完整展示人物，仅用手部、后颈、衣角、脚步等局部完成叙事，留下想象空间 | 暧昧／紧张等待／犯罪细节 |
| B13 | **群像割裂构图** | 多人同框但彼此被柱子、门框、光区或景别切开，表现关系疏离与阵营分裂 | 家庭冷战／团队不和／派系对立 |
| B14 | **曲面扭转构图** | 借助鱼眼空间、弧形建筑、弯曲镜面制造轻微空间扭曲，表达精神不安与现实失真 | 噩梦／精神崩溃／科幻异变 |
| B15 | **逆光剪影群构图** | 多人物在强逆光下形成层次剪影，仅保留轮廓和姿态，营造史诗感或匿名感 | 帮派登场／诀别／集体命运 |
| B16 | **顶部压顶构图** | 画面上方巨大天花、桥体、山体或机械结构下压，主体位于下部狭小空间，强调命运重压 | 灾难预兆／宿命感／末日前夕 |

## C · 基础通用构图（16种）

> 电影语言基本功，适合任何场景的基础构图层。运镜大师应优先掌握。

| # | 名称 | 提示词 | 适配情节 |
|:--:|:----|:-------|:--------|
| C01 | **居中对称构图** | 主体居中、左右均衡、轴线稳定、仪式感，建筑或走廊强化秩序 | 出场亮相／权威塑造／宿命感 |
| C02 | **三分法构图** | 主体落于三分交点、画面均衡自然、视线流动舒适 | 日常叙事／人物行进／风景情绪 |
| C03 | **引导线构图** | 道路/栏杆/光线汇聚主体、强化视线牵引与空间方向 | 追逐／发现目标／出发场景 |
| C04 | **对角线构图** | 主体沿对角线展开、增强张力、速度与不稳定感 | 冲突／奔跑／危险预警 |
| C05 | **框架式构图** | 门窗/洞口/前景形成内框、突出主体与窥视感 | 偷看／回忆／被困处境 |
| C06 | **留白构图** | 大面积空白衬托主体、突出孤独、宁静与心理空间 | 告别／等待／冥想／悬念 |
| C07 | **前景遮挡构图** | 利用人群/物体虚化遮挡，增强真实感、现场感与窥视视角 | 跟踪／观察／秘密交流 |
| C08 | **层次纵深构图** | 前中后景明确、景别递进、空间沉浸感强 | 群像调度／城市街景／关系铺陈 |
| C09 | **低机位仰拍构图** | 从下向上拍摄、放大气场与压迫感、天空作背景 | 英雄登场／反派威压／胜利时刻 |
| C10 | **高机位俯拍构图** | 从上向下俯视、强调渺小、孤立、秩序或混乱 | 失落／案发现场／群体场面 |
| C11 | **过肩镜头构图** | 以前景肩部建立视线关系，强化对话代入感 | 对峙／谈判／情感交流 |
| C12 | **双人平衡构图** | 两主体视觉重量均衡，体现关系张力与对等状态 | 对话／搭档／分歧与默契 |
| C13 | **单人特写构图** | 聚焦面部细节与微表情，突出情绪爆发和心理变化 | 哭泣／决断／震惊 |
| C14 | **环境人像构图** | 人物与场景共同叙事，环境信息服务角色身份 | 职业刻画／旅途／城市故事 |
| C15 | **运动跟拍构图** | 主体相对清晰、背景产生速度拖影，营造动势 | 奔跑／骑行／追车／行动场面 |
| C16 | **负空间极简构图** | 主体很小、环境占比大、强调疏离与设计感 | 孤独／科幻感／品牌感画面 |

## D · 动作戏剧构图（16种）

> 适用于武打/追逐/枪战/爆炸等动作场景。排期会中与武术大师联合使用。

| # | 名称 | 提示词 | 适配情节 |
|:--:|:----|:-------|:--------|
| D01 | **弧线包抄构图** | 多人沿弧形走廊/环路径向主体推进，主体偏位，弧线运动线形成包抄、压迫和战术推进感 | 包围／突袭推进／室内清剿 |
| D02 | **断层跃迁构图** | 主体跨越平台/楼梯/废墟间的明显垂直落差，强调高度差与动作悬停瞬间 | 逃生飞跃／灾难幸存／极限突破 |
| D03 | **交叉火线构图** | 前后左右多方向动作线交叉于主体周围，枪火、车灯、肢体、视线形成十字冲突 | 枪战中心／混乱突围／伏击反杀 |
| D04 | **甩尾切入构图** | 高速运动主体从画面边缘锐角切入，留下拖影或烟尘，主体只占小面积但驱动全画幅速度感 | 车辆追逐／紧急转弯／突现追杀 |
| D05 | **坠物压境构图** | 碎片、火花、坠落物从上方压向前景人物，下方人物闪避或反击，造成生死瞬间压迫 | 爆炸冲击／建筑坍塌／战场突变 |
| D06 | **双轴冲撞构图** | 两拨人物从不同方向高速靠近，冲突在中轴交叉处爆发，横纵双轴形成碰撞核心 | 街头冲撞／近距离格斗／正面交锋 |
| D07 | **掩体探身构图** | 主体以墙角/车门/柱子/废墟为掩体，上半身探出，动作紧张，敌人方向留出空间强化谨慎推进 | 巷战搜索／狙击警戒／突击前观察 |
| D08 | **旋身反制构图** | 捕捉身体旋转、反手制敌的瞬间，衣袍四肢离心展开，以旋转势能组织画面 | 近战反制／武术对决／打斗高潮 |
| D09 | **爆点后撤构图** | 爆炸/冲击在中后景，主体向镜头/斜后方撤退，气浪碎片火光从后追来，造成逃离紧迫感 | 爆炸逃生／任务失控／死里逃生 |
| D10 | **窄巷夹击构图** | 主体在窄巷中央，前后均有敌力形成夹击态势，墙壁和杂物压缩运动线 | 帮派伏击／夜巷追捕／绝境对峙 |
| D11 | **临空摆荡构图** | 人物悬空吊挂于绳索/机械臂/栏杆/残骸，肢体伸展+背景高度落差制造失重惊险感 | 高空逃生／潜入行动／坠落救援 |
| D12 | **回身格挡构图** | 主体转身瞬间完成格挡，攻击者从盲侧切入，利用身体扭转和手臂力线构建危机反应 | 偷袭反应／夺刀拆解／贴身体缠 |
| D13 | **烟幕突穿构图** | 主体从浓烟/蒸汽/粉尘中高速突破而出，先出现部分剪影，背后光源切穿烟层制造未知威胁 | 火线突破／战术突击／终极出场 |
| D14 | **碎面追影构图** | 利用破玻璃/金属碎片/水滴/碎镜反射动作主体的多重碎片，真实主体与碎片影像共同叙事 | 破窗入侵／身份撕裂／追踪误判 |
| D15 | **错位扑袭构图** | 前景与中景动作主体在透视中形成即将重叠的错位扑击，利用空间压缩制造突袭感 | 伏击扑杀／救援拦截／近距压制 |
| D16 | **尾迹疾撤构图** | 主体快速撤出画框，仅留下尾光/溅步/翻动的衣角/弹壳轨迹，主体半出画，强调残劲余势 | 撤退掩护／任务撤离／夜战收尾 |

## E · 高级电影感构图（16种）

> 提升画面质感与叙事深度的进阶构图，适合名场面/情绪高潮/意境镜头。

| # | 名称 | 提示词 | 适配情节 |
|:--:|:----|:-------|:--------|
| E01 | **消失点压缩构图** | 利用透视线强烈收束，空间向单一消失点挤压，突出前景张力与远端不确定感 | 追逐逃亡／宿命降临／无路可退 |
| E02 | **侧翼失重构图** | 主体偏向画面一侧边缘，另一侧大面积空间留白，制造重心外倾与失衡感 | 心理崩塌／危险逼近／孤立无援 |
| E03 | **逆向视差构图** | 前景与背景以相反方向运动，强化空间层次与时间错位感 | 时空错位／记忆闪回／命运交错 |
| E04 | **双窗对位构图** | 两个窗口并置或对称，框内人物形成呼应、对照或隔阂关系 | 隔阂对峙／秘密窥探／情感呼应 |
| E05 | **中轴破缺构图** | 打破传统中轴线稳定，通过偏移或断裂制造视觉张力与秩序崩解感 | 信念动摇／秩序瓦解／内心冲突 |
| E06 | **顶光视岛构图** | 强烈顶光将人物与环境隔离，形成孤立的光域，强化孤独与压迫感 | 审讯对峙／绝望时刻／灵魂独白 |
| E07 | **曲廊回勾构图** | 利用弧形通道或连续曲线引导视线回勾至人物，形成环绕式包裹 | 潜入探索／命运轮回／难以逃脱 |
| E08 | **前后景失焦反转** | 刻意让前景虚化、背景清晰，颠倒常规聚焦逻辑，制造陌生与不安 | 记忆错置／梦境感／心理错乱 |
| E09 | **低饱和静爆构图** | 整体低饱和色调，静止构图中隐藏情绪爆点，压抑中蓄势待发 | 情绪压抑／危机前夜／内心爆发 |
| E10 | **遮挡递进构图** | 多层遮挡由近及远递进，层层推进视线至核心主体 | 跟踪观察／步步逼近／秘密揭露 |
| E11 | **斜雨切片构图** | 斜向雨线切割画面，形成节奏分割与情绪切片，增强动感与冷感 | 雨夜戏／心事倾泻／情绪转折 |
| E12 | **冷暖断层构图** | 画面被冷暖两色分割，形成明确色温断层，象征冲突与分裂 | 立场对立／内心撕裂／伦理抉择 |
| E13 | **远端微主体构图** | 主体置于极远处，占据画面极小比例，强调渺小与环境压倒性 | 旅程启程／孤独无助／宏大叙事 |
| E14 | **纵深回望构图** | 利用长甬道或连续空间，人物回望镜头，强化空间延伸与情感牵引 | 离别回眸／犹豫不决／情感牵挂 |
| E15 | **多点视线牵引** | 多个角色视线汇聚或指向不同方向，形成多重视线牵引与焦点冲突 | 群体博弈／真相逼近／复杂关系 |
| E16 | **边缘悬停构图** | 主体处于画面边缘或临界位置，呈现悬停状态，强化不确定与紧张 | 生死抉择／悬崖边缘／命运悬念 |
