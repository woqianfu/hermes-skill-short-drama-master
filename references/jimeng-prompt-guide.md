# 即梦AI提示词公式参考

> 来源: 即梦AI提示词手册(2026.04) + 即梦操作手册(2026.04)
> 短剧大师映射: 附录S → 附录J.3 五段式提示词

## 基础公式: 5要素

```
主体 + 场景 + 动作/细节 + 风格 + 氛围/画质
```

### 反例对照

| 要素 | ❌ 模糊写法 | ✅ 量化写法 |
|------|------|------|
| 主体 | "一个老人" | "白发老者，灰色粗布衣，手掌粗糙" |
| 场景 | "室外" | "古井旁，黎明前灰蓝雾霭，石桥远景" |
| 动作 | "打水" | "弯腰将木桶沉入水中，手臂青筋微凸" |
| 风格 | "好看一点" | "国风3D高清渲染，写实质感" |
| 氛围 | "光线好" | "冷青蓝调，暖金侧光从东方渗入" |

## 进阶公式: 6维度

```
主题(主体+核心表达) + 场景(环境+时间) + 风格(艺术流派) + 色调(主色+辅色) + 构图(视角+比例) + 细节(材质/光影/动态)
```

## 四个优化原则

1. **从大到小**: 先写主体场景，再补细节光影
2. **拒绝堆砌**: 每段3-7个关键词，超过则风格模糊
3. **精准修饰**: "4K quality" > "高清"; "side light" > "强光"
4. **中英混合**: 中文定内容，英文定技术参数

## 模糊词→量化词

| 模糊 | 量化 |
|------|------|
| 好看 | 4K quality / cinematic color grading |
| 动作快 | fast movement 0.5s / rapid action |
| 画面亮 | warm golden hour / 30% highlight |
| 画面暗 | low key lighting / 70% shadow |
| 真实感 | photorealistic / PBR textures |
| 震撼 | epic scale / volumetric atmosphere |

## 5层校验

层1主体存在→层2动作对应→层3空间一致→层4风格匹配→层5约束遵守
调试顺序: 1→2→3→4→5, 层1失败=本质问题, 层5失败=叠加改进

## 参考图三原则

1. 质量优先: 光影清晰/主体突出
2. 分工明确: 每张标注用途(形态/材质/光影)
3. 迭代替换: 效果差先换参考图, 非改全部指令
