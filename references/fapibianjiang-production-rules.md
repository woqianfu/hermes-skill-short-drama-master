## 命名铁律

**前缀固定**: `发配边疆：罪妻`
  - 借势22亿IP「发配边疆」系列
  - 所有作品必须以此前缀开头，不能省略或改动

**副标题规则**:
  - ✅ 必须是**视觉冲击型钩子** — 让人一眼看到画面
  - ❌ 不要双关语 (如「制皂」太绕、太费解)
  - ❌ 不要正经 (如「改变边疆」太正、没有钩子感)
  - ❌ 不要依赖背景知识 (如「九年义务教育」需要理解穿越梗才能懂)

**示例**:
  - ✅ `发配边疆：罪妻把边关炸了` (化学老师用火药炸边关，视觉冲击)
  - ✅ `发配边疆：罪妻的实验室` (化学实验场景，视觉)
  - ❌ `发配边疆：罪妻制皂` (双关，太绕)
  - ❌ `发配边疆：罪妻改变边疆` (太正经)

## 小云雀提交流程

> ⚠️ 用户硬性要求，不允许用CLI操作小云雀

```
步骤 ①: 走短剧大师 E01 审查
  → 在短剧大师内完成第一集剧本审查 + 问题修复
  → 确认剧本通过 SQI 质量关卡

步骤 ②: 用户亲眼看着网页操作 (禁止CLI)
  → 用户必须亲自在小云雀网页端操作
  → Agent 只能提供操作指引，不能代操作
  → 不允许通过 CLI/API 提交

步骤 ③: 模型锁定
  → 必须使用 seedance2.0_mini 模型
  → 不经用户同意不得替换模型

入口路径: 短剧Agent2.0 → 上传剧本 → 出片
```

### Agent 职责
- 负责 E01 审查 + 修复剧本
- 提供操作指引（告诉用户点哪里、填什么）
- 出片后验收质量
- ❌ 不可以代操作、CLI 提交、API 调用
