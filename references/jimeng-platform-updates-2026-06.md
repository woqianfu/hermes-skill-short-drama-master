# 即梦平台功能更新（2026-06）

> 2026年6月日常扫描发现的新平台能力。记录于此供后续参考。

## 1. 运镜幅度三档控制 🆕

**来源**: 即梦官方更新 / 飞书文档
**能力**: 即梦AI视频新增移镜、摇镜，并支持三档运镜幅度设定（弱/中/强）

| 幅度 | 说明 | 适用短剧场景 |
|------|------|------|
| 弱 (Low) | 镜头运动轻微，几乎不可感知 | 日常对话、情绪克制镜 |
| 中 (Medium) | 镜头运动适度，有明显动感 | 武打L2、追逐、冲突场面 |
| 强 (Strong) | 镜头运动剧烈，冲击力强 | L3至强对决、觉醒名场面 |

**Promp注入方式**:
```
【主运镜】移镜，中幅度，从左至右，3秒完成
camera movement intensity: medium
```

**与J.3.2现有速度术语的关系**: 速度(慢/匀/快)控制运镜速率，幅度(弱/中/强)控制运镜总量。两者正交可组合：
- 慢速+弱幅度 = 细腻情绪推镜
- 快速+强幅度 = 急促甩镜

## 2. Seedance 2.0 多参考文件扩大

**来源**: 火山引擎官方

## 4. 镜头库（Shot Library）概念

**来源**: OpenCreator运镜控制方法论
**核心主张**: 把一组"能稳定出片且能出单的镜头原子"固定下来（hero环绕/细节摇镜/揭示推镜/使用中景/对比镜），后续项目只替换输入（产品/人物/场景brief），不再重写运镜提示词。
**与短剧大师关联**: 附录K已有批量生产理念，镜头库是进一步的模板沉淀。建议在下一轮项目启动时建立项目专属镜头库。
