## 核心差异：付费 vs 免费模式

| 维度 | 付费模式 | 免费模式 |
|------|------|------|
| 盈利驱动 | 单点悬念→用户付费解锁 | 观看时长+播放量→平台分账 |
| 钩子策略 | 每集结尾一个付费点 | 高密度钩子矩阵（开篇/中段/结尾） |
| 留存手段 | 悬念驱动"下一集" | 情绪满足+新期待循环 |
| 节奏要求 | 集末爆点 | 每集3秒抓眼+3分钟入戏+结尾留钩子 |
| 用户行为 | 主动付费→追更 | 划屏速度1.5s/条→前5秒决定留存 |

## 钩子矩阵三阶段布局

### 开篇钩子（3-5秒）
- **不铺垫、不绕弯**原则
- 甜虐情感类：极致痛感开场（如手持流产单+离婚协议）
- 爽文逆袭类：藏身份反套路（10秒内"受辱-藏锋"反转）
- 喜剧类：无厘头反差破局
- **关键指标**：前5秒留存率≥55%（行业爆款基准线）

### 中段钩子（每集三件套）
```
新冲突 → 呼应旧伏笔 → 情绪爆点 → 新悬念产生
```
- 每集都有信息增量，推动剧情向主线靠拢
- 冲突-情绪爆点形成闭环，独立可看
- 观众中途退出也能快速回归

### 结尾钩子（三类形式）
| 类型 | 触发方式 | 用户情感 |
|:--:|------|:--:|
| 悬念式 | 隐藏身份/令牌暗示 | "快亮明身份" |
| 目标推进式 | 完成阶段目标→新目标明确 | "继续追" |
| 危机式 | 新威胁降临 | "如何化解" |

## 节奏黄金公式（3+3+钩）

```
3秒抓眼：强冲突开场 → 前情/人物关系在画面中自然交代
3分钟入戏：冲突升级 → 主角反击意识觉醒 → 每30秒情绪波动
每集留钩子：结尾→新冲突/反转/悬念 → 紧张-缓解-新紧张循环
```

## 结构模型：目标拆解+主线串联

```
全剧核心目标
  ├─ 阶段目标1（E01-E03）达成 → 成就感
  ├─ 阶段目标2（E04-E06）达成 → 成就感
  ├─ 阶段目标3（E07-E10）达成 → 成就感
  └─ 主线伏笔贯穿全程 → 追更驱动力
```

- 每2-3集完成一个小目标
- 主线伏笔保持剧情连贯性
- 单集可独立观看 + 全剧追主线
