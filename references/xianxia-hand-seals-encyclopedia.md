## 短剧战斗场景映射

### L2 超凡战斗 → 结印组合

| 战斗阶段 | 推荐结印 | 用途 |
|---------|:--:|------|
| 起手/戒备 | 合掌印 或 三角印 | 感知敌人/启动结界 |
| 首次攻击 | 剑指印 或 开掌印 | 破邪/灵力释放 |
| 敌方压制 | 交叉印 或 镇地印 | 封印/禁锢 |
| 受伤回血 | 莲华印 | 治疗净化 |
| 蓄力大招 | 同心印 或 乾坤印 | 核心攻击/操控天地 |
| 终极防御 | 天门印 或 灵塔印 | 护体屏障 |

```
三角印(感知)→剑指印(试探攻击)→交叉印(封印反击)→
同心印(蓄力)→乾坤印(天地之力)→锁魂印(最终禁锢)
```

配合五维打斗公式中的「物理受力」维度：结印手势变化速度 = 战斗节奏。
