# AI短剧分镜Skill（storyboard-lite·小阳哥）— 附录M

> 来源：小阳哥开源短剧分镜Skill（2026）。GitHub: towardsyoung/video-agent-skills

## 输入输出
| 输入 | 必需/可选 | 默认值 |
|------|:---------:|--------|
| 剧本正文 | ✅必需 | — |
| 故事类型 | 可选 | 都市职场 |
| 美术风格 | 可选 | 真人现代都市 |
| 输出格式 | 可选 | Markdown |

## 12种故事类型
`都市职场` `甜宠恋爱` `悬疑惊悚` `热血动作` `仙侠玄幻` `家庭温情` `喜剧幽默` `科幻末世` `历史史诗` `心理剧情` `恐怖超自然` `成长剧情`

## 11种美术风格
`真人现代都市` `真人古装中国` `真人武侠` `2D国风` `2D日漫` `2D扁平` `3D动画` `3D国风` `3D国风赛博` `3D黏土定格`

## 分镜表14字段
| # | 字段 | 说明 |
|:-:|------|------|
| 1 | 序号 | 镜号（1-17） |
| 2 | 画面描述 | 简洁视觉描述 |
| 3 | 场景 | 地点 |
| 4 | 关联资产名称 | 角色+道具 |
| 5 | 时长 | 秒（按M.6公式计算） |
| 6 | 景别 | 远景/全景/中景/近景/特写/鸟瞰 |
| 7 | 运镜 | 固定/缓慢推进/跟拍/摇移/俯拍 |
| 8 | 角色动作 | (开篇)或(承接上镜:...) |
| 9 | 朝向 | 角色面朝方向 |
| 10 | 空间关系 | 多人镜角色位置 |
| 11 | 情绪 | 角色情绪状态 |
| 12 | 台词 | 逐字复制对白 |
| 13 | 音效 | 仅物理声源/环境声，不写BGM |

## 时长计算公式
| 情绪/语速 | 公式 |
|-----------|------|
| 愤怒/急促 | 台词字数÷4 + 1s |
| 正常对话 | 台词字数÷3 + 1s |
| 悲伤/耳语 | 台词字数÷2 + 1s |

## 工作流
```
剧本文字 → storyboard-lite(自动提取资产+14字段分镜表)
→ 人工检查分镜逻辑/景别衔接/时长合理性
→ Markdown分镜表→即梦/小云雀→逐镜生成
→ 剪映组装→字幕特效→调色音效→成片
```
