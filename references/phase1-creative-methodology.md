# 短剧大师 Phase 1 创作方法论速查

## 6 步工作流

```
第1步 定位赛道 → 第2步 设计人物 → 第3步 打造第一屏
  → 第4步 规划全剧 → 第5步 写正文 → 第6步 自检
```

## 赛道速查（11 条）

女频：霸总 / 追妻 / 重生复仇 / 甜宠 / 萌宝
男频：战神 / 赘婿 / 神豪 / 修仙玄幻
通用：年代家庭 / 古装权谋

## 角色功能位（5 类）

主角 / 女主（互利→守护→并肩）/ 第一反派 / 终极反派 / 成长伙伴

## 压迫系统公式

上位压迫者 → 可见压迫 → 主角反应 → 升级 → 更大压迫

## 第一屏公式

E01 羞辱+信号 → E02 后果+证据 → E03 反击+升级
→ E04-E06 第二轮 → E07-E09 第三轮 → E10 落幕+遗产+尾钩

## 钩子密度

每集 ≥1 信息揭露 / 每 2-3 集一次能力释放
前 3 集必须出现视觉信号 / E01 尾钩必须悬念

## 正文铁律

一景一事 / 对白不解释 / 冲突前置 / 尾钩回收 / 信息渐进 ≤7/集
单句 ≤25 字 / 同角色固定称呼 / 保留反讽和潜台词
