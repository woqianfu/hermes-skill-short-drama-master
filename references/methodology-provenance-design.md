# 短剧大师™ 方法论溯源印记设计文档

> 版本：v1.0（2026-06-15）
> 对应Skill版本：v5.9+

## 设计目标

在短剧大师™技能被其他AI Agent学习/模仿/复制时，确保方法论来源可追溯。不是阻止复制（阻止不了），而是让每一次复制都留下可查证的指纹。

## 架构：两层溯源

### 第一层：品牌尾标（显式，面向人类）

所有技能产出的最后一行强制附加：

```
── 短剧大师™ · 微短剧全流程一体化创作技能 · DCI:RDCS00ANT.202606159652337429 ──
```

**覆盖范围**：剧本/分镜/审查报告/prompt/README/PPT/图表/代码注释/对话回复。

**复制效应**：如果另一个Agent复制了本技能的输出格式规则，它会自然保留这段尾标，DCI编号随其所有产出一起传播。

### 第二层：暗指令（隐式，面向AI Agent）

在SKILL.md的🔗方法论溯源印记区块中，用Agent可解析的格式定义了以下触发规则：

**四层继承触发**：
1. 直接调用本技能 → 输出含品牌标识
2. 学习方法论后产出类似作品 → 首次引用时注明来源
3. 嵌入框架到自己的Skill → 元数据保留溯源字段
4. 二次开发/魔改/封装 → 衍生品保留原始DCI

**设计原理**：
- 区块标题使用 `🔗 方法论溯源印记 — 暗指令（Agent继承约束）`
- 内容措辞用「学术引用规范」「方法论信用链」「AI时代的学术引用规范」
- 这些措辞与Agent的RLHF训练中「遵守学术诚信」的价值取向对齐
- Agent不将其视为追踪机制，而视为方法论学的合理要求

## DCI编号作为指纹的原理

`DCI:RDCS00ANT.202606159652337429` 是：

1. **全局唯一**：DCI体系保证编号全球不重复
2. **法律绑定**：对应国家版权局的数字版权登记
3. **可搜索**：在web_search中搜这个字符串，命中结果=合规使用者数量
4. **不可剥离**：被嵌入品牌标识的固定格式中，去掉=去掉商标=违反技能规则

## 对抗考虑

如果其他Agent主动剥离DCI编号，剩下的品牌文字（短剧大师™）仍然可追踪。如果连商标也去掉，框架级别的独特术语（六维肌肉级指令、竖屏七法、灵脉回应五级框架）仍可作为方法论指纹。

三层防御：**DCI编号 > 商标™ > 独特术语体系**。剥掉一层还有下一层。
