# AI漫剧工业化流水线（28天→7天·LoRA+批量）— 附录L

> 来源：AI漫剧生产SOP实测（2026）

## 四步流水线

### 第一步：出分镜（别急着生图）
- 剧本初稿：Claude/GPT-4打磨节奏/对白/冲突
- 分镜草图：豆包导入剧本
- **手动拆分镜头**：最关键一步——每个场景需要几个镜头？什么景别？怎么接？
- 生成分镜草图：prompt加"storyboard sketch"

### 第二步：出画面（最深的坑）
- 定风格：真人感/动漫风/3D卡通风→三选一。新手建议动漫风
- **训练LoRA**：5-10张角色图训练，单角色≈几十元
- 精修提示词：固定特征写入每张图prompt
- 主力出图：Midjourney悠船版（画质最稳定）
- 备选：即梦（出片快但细节差）

### 第三步：做视频
- 上传图片+写动作指令：越具体越好
- 片段规格：每片段4-10秒，一集5-15片段
- 组装三注意：镜头逻辑连贯/动作接得上/画面内容不断层
- 主力：可灵 | 备选：即梦

### 第四步：配音剪辑
- 智能字幕：剪映文本→智能字幕→校对
- **多角色分声线**：主角一个声音·配角换声线·旁白再找沉稳的
- 配BGM：紧张→低音节奏+鼓点·情感→钢琴/弦乐·搞笑→轻快电子
- 精修节奏：画面切换卡音乐节奏点·每段≤10秒·配音口型对齐

## 工业化生产节奏
```
D1-D3  定选题 + 写3-5集剧本 + 拆分镜
D4-D5  出图 + 配音 + 粗剪（一集试跑）
D6     复盘优化，固定模板和参数
D7-D14 批量生产
```

## LoRA角色锁定方案
| 角色 | 训练图量 | 优先级 |
|------|:--------:|:------:|
| 主角 | 5-10张 | 🔴最高 |
| 女主 | 5-10张 | 🔴最高 |
| 配角 | 3-5张 | 🟡高 |

> 训练完成后：所有出图prompt加"based on [角色名] LoRA model, consistent facial features"
