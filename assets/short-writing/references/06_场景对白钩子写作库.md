# 场景、对白与钩子写作库

## 单场戏与竖屏可拍性

## 1. 单场戏最低配置

```text
一个目标：人物当场想得到什么。
一个阻碍：谁不让他得到，凭什么不让。
一个信息差：谁知道秘密，谁误判。
三次升级：语言、动作、证据、旁人站队逐步加压。
一次转折：权力、关系或信息发生变化。
一个钩子：下一场必须处理的新问题。
```

## 2. 竖屏场景语法

| 写作对象 | 竖屏处理 | 剧作价值 |
| --- | --- | --- |
| 人脸 | 近距离表情变化 | 误会、忍耐、崩溃、反击 |
| 手部动作 | 抢手机、撕合同、递鉴定、按录音 | 证据和权力变化 |
| 站位 | 一人被围、两人对峙、旁人站队 | 关系压迫可视化 |
| 小空间 | 电梯、车内、病房、饭桌、办公室 | 低成本高压迫 |
| 公开场 | 婚礼、寿宴、直播、公司大厅 | 羞辱和审判放大 |

## 3. 场景升级动作

逼问、夺物、站队、公开、切断。每场至少使用两种，否则容易变成说明戏。

## 对白、情绪爆发与金句

## 1. 台词功能表

| 功能 | 句子目的 | 写法方向 |
| --- | --- | --- |
| 逼问 | 迫使对方回答事实 | 把旧账、证据和称谓压进一句话 |
| 威胁 | 让对方付出代价 | 说明对方最怕什么被公开 |
| 试探 | 用半真半假钓秘密 | 问对方不该知道的信息 |
| 挑拨 | 改变旁人站队 | 把私人冲突变成舆论判断 |
| 反击 | 改变权力关系 | 不求信任，只宣布后果 |
| 追悔 | 承担具体伤害 | 说出当初做错的动作和代价 |

## 2. 情绪爆发阶梯

1. 忍：不解释，只看对方如何继续压迫。
2. 刺：用一句具体旧账反刺。
3. 揭：拿出证据或指出漏洞。
4. 断：做出不可逆选择。
5. 爆：公开判决关系或身份。

## 3. 金句生成方法

```text
关系：你是我丈夫 / 你是我妈 / 你是我救过的人
伤害：你信了别人 / 你逼我认错 / 你拿孩子威胁我
立场：我不再解释 / 我不再让 / 我亲自公开
```

## 钩子、反转与桥段

## 1. 钩子强度分级

| 强度 | 形式 | 说明 |
| --- | --- | --- |
| 弱钩子 | 明天再说、他来了 | 问题不清或代价不明 |
| 中钩子 | 证据出现但未打开 | 观众知道等什么 |
| 强钩子 | 证据将揭 + 关系代价 | 下一集会改变站队 |
| 追更钩子 | 将赢未赢、将认未认、将公开未公开 | 情绪最强，适合追更点 |

## 2. 七类结尾钩子

身份、证据、关系、危险、选择、舆论、反派升级。

## 3. 反转设计表

```text
观众以为：
主角以为：
反派以为：
真实情况：
反转后谁更危险：
反转后谁改变站队：
下一集必须处理什么：
```

## 4. 桥段迁移规则

保留功能，不保留原剧情。把公开羞辱、证据露一半、身份错认、亲子鉴定、直播曝光等机制换成人物关系、证据物件和场景空间。

| 对白类型 | 作用 | 写法 |

## 场景写作总原则

短剧场景要用动作承载信息。能用抢手机表达的，不用三段解释；能用站队表达的，不用旁白说明；能用公开场表达的，不写两个人平静聊天。

| 场景元素 | 必须明确 | 不合格表现 |
| --- | --- | --- |
| 目标 | 主角当场想拿到什么 | 只是聊天 |
| 阻碍 | 谁不让他得到，凭什么 | 对方只是不高兴 |
| 信息差 | 谁知道真相，谁误判 | 所有人知道的信息一样 |
| 升级 | 语言、动作、物件、旁观者依次加压 | 从头到尾同一强度 |
| 转折 | 权力、证据、关系或目标改变 | 结尾只是停顿 |
| 钩子 | 下一场必须回答的问题 | 下集继续看就知道 |

## 场景、对白、钩子与可拍表达

## Agent 角色设定

短剧场景对白教练型 Agent，负责把剧情节点写成短、狠、可演、能转折的场次。

工作目标：
- 每场戏先定义目标、地位差、信息差、禁忌和结尾转折。
- 对白必须服务羞辱、试探、反击、摊牌、误判、追悔或保护等功能。
- 台词短、直接、有动作压力，但不把解释当对白。
- 每场结尾改变权力、信息、关系或行动方向。
- 正文格式必须符合第几集/场次/场景/人物/动作/对白。

禁止做法：
- 不要用抽象评价代替场景级判断。
- 不要把类型公式当成固定剧情模板。
- 不要复用本地完整剧本的人物、场景、对白或连续桥段。
- 不要在缺少信息时停在追问；先给合理假设并继续输出可执行版本。
- 不要让正文变成小说梗概；正文必须能被演员和导演直接理解。

## 标准工作流

| 步骤 | Agent 要判断什么 | 产出 |
| --- | --- | --- |
| 场景定功能 | 判断本场是压迫、反击、试探、摊牌、追悔、认亲还是转场。 | 场景功能。 |
| 定人物目标 | 每个角色此刻想得到什么。 | 角色策略。 |
| 定地位差 | 谁掌握资源、证据、名分、场地、旁观者。 | 压迫关系。 |
| 定不能说 | 谁有什么真相不能公开。 | 潜台词。 |
| 写动作先行 | 先写可拍动作，再写台词。 | 动作-对白链。 |
| 压缩台词 | 删解释、删重复、删心理说明。 | 短句和反应。 |
| 收尾留钩 | 让场景结尾提出下一场问题。 | 结尾钩子。 |

执行细则：
1. 场景定功能
   - 判断：判断本场是压迫、反击、试探、摊牌、追悔、认亲还是转场。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：场景功能。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
2. 定人物目标
   - 判断：每个角色此刻想得到什么。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：角色策略。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
3. 定地位差
   - 判断：谁掌握资源、证据、名分、场地、旁观者。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：压迫关系。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
4. 定不能说
   - 判断：谁有什么真相不能公开。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：潜台词。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
5. 写动作先行
   - 判断：先写可拍动作，再写台词。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：动作-对白链。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
6. 压缩台词
   - 判断：删解释、删重复、删心理说明。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：短句和反应。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
7. 收尾留钩
   - 判断：让场景结尾提出下一场问题。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：结尾钩子。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。

## 诊断表

| 问题 | 表面症状 | 对应修复动作 |
| --- | --- | --- |
| 对白解释剧情 | 角色互相说已知信息。 | 把信息变成争夺物或证据物。 |
| 没有场景目标 | 大家只是聊天。 | 给角色一个必须拿到的结果。 |
| 互骂无推进 | 台词狠但状态不变。 | 每轮攻击后改变证据、站队或行动。 |
| 主角只辩解 | 主角一直解释自己没错。 | 让主角抓一个事实反问或行动反击。 |
| 反派只放狠话 | 威胁没有实际筹码。 | 给反派合同、视频、孩子、职位或舆论筹码。 |
| 潜台词缺失 | 人物把真实想法全说完。 | 让角色说眼前事，藏真正伤口。 |
| 场景结尾平 | 没有下一集问题。 | 用来人、电话、证据、身份露角收尾。 |
| 格式不统一 | 像小说段落。 | 改成场次、人物、动作、对白分行。 |

逐项诊断写法：
### 诊断 1：对白解释剧情

- 诊断信号：角色互相说已知信息。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：把信息变成争夺物或证据物。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 2：没有场景目标

- 诊断信号：大家只是聊天。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：给角色一个必须拿到的结果。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 3：互骂无推进

- 诊断信号：台词狠但状态不变。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：每轮攻击后改变证据、站队或行动。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 4：主角只辩解

- 诊断信号：主角一直解释自己没错。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：让主角抓一个事实反问或行动反击。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 5：反派只放狠话

- 诊断信号：威胁没有实际筹码。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：给反派合同、视频、孩子、职位或舆论筹码。

## 常见失败

| 失败类型 | 后果 | 修正 |
| --- | --- | --- |
| 只讲理论 | 用户拿不到可写内容。 | 每段理论后补一个结构动作或输出字段。 |
| 直接写正文 | 还没确定类型和第一屏。 | 先做任务判断和类型承诺。 |
| 案例复刻 | 新稿像某一部剧。 | 至少混合三个机制，并列必须替换项。 |
| 压力不可见 | 观众不知道主角为何必须行动。 | 改成公开场、可见损失、旁观者站队。 |
| 反击工具缺失 | 主角只靠忍耐和解释。 | 补证据、身份、资源、技能、孩子线索或规则漏洞。 |
| 尾钩模糊 | 每集结尾没有下一集问题。 | 用证据、来人、电话、身份露角或关系选择收尾。 |
| 对白说明化 | 人物替作者解释设定。 | 让信息成为争夺物。 |
| 中段重复 | 同类争吵循环。 | 升级压力来源和裁判层级。 |
| 结尾无清算 | 真相公开但情感债没还。 | 补承认、补偿、放权、公开承担。 |
| 格式混乱 | 正文像小说或分镜秒表。 | 统一剧本格式。 |

## 自检清单

- [ ] 是否先判断了用户任务，而不是直接写？
- [ ] 是否明确了主类型和主追更情绪？
- [ ] 第一屏是否有公开压力、可见损失和旁观者站队？
- [ ] 主角是否有可重复使用的反击工具？
- [ ] 反派或压迫者是否掌握具体资源、规则或证据？
- [ ] 每一集是否有事件，而不是只有情绪对话？
- [ ] 每一集结尾是否留下具体下一集问题？
- [ ] 前3集是否完成压迫、误判和小反击？
- [ ] 前10集是否打开中段更大压力？
- [ ] 60-100集是否有分段追更点？
- [ ] 人物关系是否有可拍的关系债？
- [ ] 对白是否有目标、地位差、信息差和转折？
- [ ] 网文改编是否删除了慢铺垫和无效支线？
- [ ] 对标案例是否只借机制，不借原文表达？
- [ ] 是否列出必须替换的人物、证据、职业、场景和反转顺序？
- [ ] 正文是否使用第几集/场次/场景/人物/动作/对白格式？
- [ ] 质量自检是否指出了仍需补强的地方？
