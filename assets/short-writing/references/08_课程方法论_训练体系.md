# 课程讲师方法论深度剖析与训练体系

## 1. 方法论总判断

课程讲师的编剧方法论可以概括为：把故事创作从灵感崇拜改造成可拆解、可训练、可交付、可复盘的复杂工程。

## 2. 创作流程

```text
创作冲动
-> 故事创意五元素：人物 / 地点 / 情境 / 行动 / 主题
-> 素材卡：高光画面、人物、道具、冲突、台词、反转、场景
-> 人物小传：身份、缺陷、欲望、秘密、创伤、行动方式
-> 故事梗概 / 短剧故事大纲
-> 前10集卡点
-> 分集大纲
-> 正文
-> 多轮修改：故事 / 人物 / 单场 / 台词 / 交付
```

## 3. 短剧产品论

短剧是故事性产品。产品要解决解压、代入、黏性；剧作要提供爽点、共鸣和追更卡点。

## 4. 精彩故事三指标

- 对抗：目标利益冲突必须可见。
- 变化：权力、信息、关系、目标或规则必须变化。
- 共情：观众要能站到主角处境里，而不是只理解设定。

## 5. 训练方法

1. 每天拆一个第一屏，只记录压迫、关系、证据和钩子。
2. 每周做一次同题材前10集表。
3. 把素材卡按人物、道具、场景、秘密、反转、台词分类。
4. 每次改稿只抓一个优先级最高的问题，避免全盘重写。


## 创作训练路径

| 训练 | 任务 | 验收标准 |
| --- | --- | --- |
| 第一屏训练 | 每天拆一个第一屏 | 能说清谁压谁、压迫凭什么、钩子是什么 |
| 题材训练 | 每周做一个题材矩阵 | 能区分主赛道和副元素 |
| 前10集训练 | 同一创意写三版前10集 | 第7集有回报，第10集有强未完成 |
| 场景训练 | 把说明戏改成对峙戏 | 目标、阻碍、信息差、三次升级齐全 |
| 对白训练 | 把解释句改成功能句 | 每句承担逼问、反击、挑拨、追悔或钩子 |
| 复盘训练 | 用诊断表改一版旧稿 | 问题优先级清楚，改写可执行 |

## Agent 使用守则

1. 先判断任务，不要直接写正文。
2. 先定题材承诺，再定人物关系。
3. 先写第一屏和第10集，再补中间。
4. 先诊断场景目标，再改对白。
5. 每次输出都要有自检，不给空泛建议。


## 从方法到训练的闭环

这套方法不依赖灵感一次成型，而是把短剧创作拆成可以反复训练的模块：第一屏、人物低位、反派筹码、证据物件、前10集、单场戏、对白、长线阶段、诊断改写。

| 训练模块 | 每日练习 | 验收问题 |
| --- | --- | --- |
| 第一屏 | 拆一个开场，只写谁压谁和钩子物件 | 10秒内能否看懂冲突 |
| 题材 | 把一个想法归入主赛道和副元素 | 主承诺是否唯一 |
| 人物 | 给主角写低位和反击工具 | 主角为什么不能立刻赢 |
| 反派 | 给反派补现实筹码 | 反派凭什么压主角 |
| 前10集 | 写第1、3、7、10集卡点 | 是否有承诺、回报和追更 |
| 场景 | 把说明戏改成对峙戏 | 是否有目标、阻碍、信息差、升级 |
| 对白 | 把解释句改成功能句 | 每句是否有逼问、反击、挑拨或钩子 |
| 复盘 | 用评分表改一稿 | 优先级是否清楚 |

## 七天训练安排

```text
第1天：只练第一屏。写10个不同题材的第一屏。
第2天：只练主角低位。每个第一屏补不能立刻赢的原因。
第3天：只练反派筹码。把骂人型反派改成有现实权力。
第4天：只练证据物件。为每个项目设计3个可拍证据。
第5天：只练前10集。写第1、3、7、10集，不补中间。
第6天：只练单场戏。每场必须有三次升级和一个转折。
第7天：只练诊断。用评分表找最严重的三个问题。
```

## 素材卡库建设

| 卡片 | 记录内容 | 使用时机 |
| --- | --- | --- |
| 压迫卡 | 公开羞辱、资源封锁、关系站错 | 第一屏和前3集 |
| 证据卡 | 鉴定、录音、合同、病历、旧照片、账本 | 第3-10集和中段 |
| 关系卡 | 救命债、血缘债、感情债、金钱债、名誉债 | 人物关系和追悔 |
| 场景卡 | 饭桌、病房、公司、婚礼、车内、直播间 | 正文和拍摄判断 |
| 对白卡 | 逼问句、反击句、追悔句、挑拨句、钩子句 | 正文改写 |
| 风险卡 | 暴力、违法、擦边、迷信等替代表达 | 提案和交付 |

## 复盘问题

```text
1. 今天写的第一屏是否不用背景解释。
2. 主角低位是否可见。
3. 反派是否掌握现实筹码。
4. 第3集是否证明题材承诺。
5. 第7集是否给观众局部回报。
6. 第10集是否是强未完成。
7. 中段是否换规则而不是换骂人对象。
8. 单场戏是否有目标、阻碍、信息差和转折。
9. 对白是否在做事，而不是讲道理。
10. 提案是否让合作方看见能拍、能卖、能规避。
```

## 课程方法论、训练体系与编剧练习

## Agent 角色设定

短剧训练教练型 Agent，负责把创作方法拆成可反复训练、评估和升级的练习。

工作目标：
- 训练不是讲概念，而是让用户产出可检验的第一屏、前10集、人物压力和场景对白。
- 每个练习都要有输入、限制、输出和评分标准。
- 用案例机制做拆解训练，不让学员背剧情。
- 用改写练习修复慢铺垫、弱压迫、无反击、尾钩平等问题。
- 建立从创意到完整项目包的训练路径。

禁止做法：
- 不要用抽象评价代替场景级判断。
- 不要把类型公式当成固定剧情模板。
- 不要复用本地完整剧本的人物、场景、对白或连续桥段。
- 不要在缺少信息时停在追问；先给合理假设并继续输出可执行版本。
- 不要让正文变成小说梗概；正文必须能被演员和导演直接理解。

## 标准工作流

| 步骤 | Agent 要判断什么 | 产出 |
| --- | --- | --- |
| 基础诊断 | 让学员写一句话创意和第一屏。 | 选题问题清单。 |
| 类型训练 | 同一创意改成追妻、复仇、萌宝、战神等版本。 | 类型差异表。 |
| 前10集训练 | 每集只写事件、压力、反转、尾钩。 | 前10集表。 |
| 人物训练 | 为每个角色写功能位和关系债。 | 人物压力表。 |
| 对白训练 | 把解释性对白改成目标冲突。 | 场景改写。 |
| 案例训练 | 拆 3 个案例的共同机制。 | 机制卡。 |
| 项目训练 | 整合成完整项目包和前5集正文。 | 交付稿。 |

   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。

3. 前10集训练
   - 判断：每集只写事件、压力、反转、尾钩。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：前10集表。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
4. 人物训练
   - 判断：为每个角色写功能位和关系债。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。

逐项诊断写法：
### 诊断 1：练习太空

- 诊断信号：只让学员写感想。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：改成有具体输出和评分标准。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

## 常见失败

| 失败类型 | 后果 | 修正 |
| --- | --- | --- |
| 只讲理论 | 用户拿不到可写内容。 | 每段理论后补一个结构动作或输出字段。 |
| 直接写正文 | 还没确定类型和第一屏。 | 先做任务判断和类型承诺。 |
| 案例复刻 | 新稿像某一部剧。 | 至少混合三个机制，并列必须替换项。 |
| 压力不可见 | 观众不知道主角为何必须行动。 | 改成公开场、可见损失、旁观者站队。 |
| 反击工具缺失 | 主角只靠忍耐和解释。 | 补证据、身份、资源、技能、孩子线索或规则漏洞。 |
| 尾钩模糊 | 每集结尾没有下一集问题。 | 用证据、来人、电话、身份露角或关系选择收尾。 |
| 对白说明化 | 人物替作者解释设定。 | 让信息成为争夺物。 |
| 中段重复 | 同类争吵循环。 | 升级压力来源和裁判层级。 |
| 结尾无清算 | 真相公开但情感债没还。 | 补承认、补偿、放权、公开承担。 |
| 格式混乱 | 正文像小说或分镜秒表。 | 统一剧本格式。 |
