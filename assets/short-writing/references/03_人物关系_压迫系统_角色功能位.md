# 人物关系、压迫系统与角色功能位

## 1. 主角设计

短剧主角要先有处境，再有性格。观众不是先爱上履历，而是先站队。


## 人物系统总表

| 角色位 | 作用 | 必须拥有 | 常见失败 |
| --- | --- | --- | --- |
| 主角 | 承受压迫并主动反击 | 低位、欲望、不能赢的原因、反击工具 | 一直忍、只解释、没有行动 |
| 贴身反派 | 前10集高频制造阻碍 | 现实权力、证据、舆论或亲密关系筹码 | 只会骂人 |
| 误判关系者 | 制造情感伤害和后期追悔 | 站错队的理由、发现真相的节点、补偿行动 | 后期只道歉 |
| 资源控制者 | 中段换规则 | 公司、家族、法律、资本或舆论资源 | 只是更凶的小反派 |
| 助攻 | 递证据或制造现场 | 风险和代价 | 无条件帮忙 |
| 旁观者 | 放大公开审判 | 站队变化 | 只当背景板 |

## 主角低位类型

| 低位 | 适合题材 | 第一集表现 |
| --- | --- | --- |
| 身份低位 | 霸总、赘婿、神豪、战神 | 被当保姆、骗子、穷人、无能者 |
| 信息低位 | 悬疑、婚姻、复仇 | 不知道证据被换或亲密关系背叛 |
| 关系低位 | 追妻、虐恋、家庭 | 被最该相信自己的人误判 |
| 资源低位 | 逆袭、年代、家庭 | 钱、工作、房子、孩子安全被控制 |
| 名誉低位 | 复仇、婚姻、职场 | 被公开造谣、直播羞辱、公司群审判 |

## 反派升级阶梯

```text
1. 语言羞辱：让观众快速站队。
2. 物件压迫：合同、鉴定、病例、账本、录音、手机被抢。
3. 关系压迫：丈夫、父母、孩子、老板或家族站错队。
4. 资源压迫：断钱、封号、撤资、开除、报警、舆论。
5. 规则压迫：公司制度、家法、法律程序、平台规则。
6. 终局压迫：旧案、血缘、真正幕后、全场公开审判。
```

## 人物不是履历，而是功能

短剧人物首先承担剧作功能，其次才是性格。一个人物如果不能制造压迫、递出证据、改变站队、造成误判、推动追悔或触发反击，就应该合并、删除或改成功能明确的角色。

| 功能 | 人物必须做到 | 典型角色 |
| --- | --- | --- |
| 压迫 | 让主角第一集处于低位 | 婆婆、丈夫、老板、小三、债主、家族掌权者 |
| 误判 | 给主角最痛的情感伤害 | 爱人、父母、恩人、上司、孩子 |
| 递证 | 让信息变成可拍物件 | 助理、医生、律师、孩子、旧友 |
| 拱火 | 把私人冲突公开化 | 同事、亲戚、网友、宾客、董事会成员 |
| 背刺 | 让主角小胜后付出代价 | 闺蜜、兄弟、下属、亲戚、前任 |
| 补偿 | 让追悔成为行动 | 误判关系者、亲生父母、男主、妻子 |

## 主角设计六问

```text
1. 主角被谁压在低位。
2. 这个低位为什么现在无法立刻改变。
3. 主角最想夺回什么。
4. 主角手里有什么反击工具，但为什么不能马上用完。
5. 主角第一次主动行动是什么。
6. 终局主角的新行为和第一集有什么不同。
```

如果主角只能忍耐，没有主动行动，观众会疲劳；如果主角第一集就全赢，后续会失去追更理由。理想状态是“能赢一小口，输一大口，再换打法”。

## 误判关系者设计

| 阶段 | 行为 | 观众情绪 |
| --- | --- | --- |
| 初期 | 相信反派或伪证，伤害主角 | 愤怒，等待追悔 |
| 前10集 | 发现第一个漏洞，但因为新证据继续误判 | 更痛，等待真相 |
| 中段 | 开始查证，却付出迟到代价 | 追悔开始兑现 |
| 后段 | 公开承认具体错误，用行动补偿 | 清算或重新选择 |

追悔不能只写“我错了”。必须写错在哪里、造成什么后果、愿意付出什么代价、主角是否接受。

## 反派设计六层

| 层级 | 作用 | 升级方式 |
| --- | --- | --- |
| 嘴脸反派 | 第一集快速让观众站队 | 辱骂、嘲讽、诬陷、挑拨 |
| 物件反派 | 控制证据和当场局势 | 抢手机、撕合同、换鉴定、删监控 |
| 关系反派 | 让主角最亲近的人站错队 | 造伪证、装受害、利用旧情 |
| 资源反派 | 封锁现实出口 | 断钱、开除、撤资、报警、封号 |
| 规则反派 | 把私人冲突升级成制度压迫 | 家法、公司制度、继承规则、平台舆论 |
| 幕后反派 | 承接终局秘密 | 旧案、身世、真正受益者、隐藏操盘者 |

## 配角合并原则

```text
只负责聊天的配角，合并。
只负责解释背景的配角，改成递证据。
只负责骂人的配角，给他现实筹码。
只负责卖萌的孩子，给他线索、选择或危险。
只负责深情的男主，给他误判、行动补偿和公开选择。
```

## 人物关系、压迫系统与角色功能位

## Agent 角色设定

人物压力架构师型 Agent，负责把角色从姓名表变成能持续制造事件的功能系统。

工作目标：
- 为每个主要人物定义欲望、恐惧、误判、资源和可造成的压力。
- 设计主角低位来源和反击工具，不让主角长期被动。
- 把关系债转成可拍行动：亏欠、误会、承诺、背叛、替身、亲缘、财产。
- 让反派不是单纯作恶，而是掌握某种规则或证据。
- 安排助攻、裁判、误导者、证人、孩子、外部权威等功能位。

禁止做法：
- 不要用抽象评价代替场景级判断。
- 不要把类型公式当成固定剧情模板。
- 不要复用本地完整剧本的人物、场景、对白或连续桥段。
- 不要在缺少信息时停在追问；先给合理假设并继续输出可执行版本。
- 不要让正文变成小说梗概；正文必须能被演员和导演直接理解。

## 标准工作流

| 步骤 | Agent 要判断什么 | 产出 |
| --- | --- | --- |
| 列关系债 | 每对核心人物之间写出欠了什么、误会什么、怕公开什么。 | 关系债表。 |
| 定压迫源 | 家族、公司、婚姻、身份、舆论、法律、亲缘、阶层。 | 压力系统。 |
| 分配资源 | 谁掌握钱、证据、名分、孩子、合同、职位、病历、监控。 | 资源表。 |
| 设计误判链 | 误判从哪里来，谁强化，谁受益，谁能推翻。 | 前10集误判链。 |
| 安排站队变化 | 旁观者、亲密关系、权力方在何时改变判断。 | 站队节点。 |
| 设置反派升级 | 小反派到中层裁判再到幕后规则掌控者。 | 压迫阶梯。 |
| 校验主角行动 | 每 1-2 集主角必须做一次主动选择。 | 行动清单。 |

执行细则：
1. 列关系债
   - 判断：每对核心人物之间写出欠了什么、误会什么、怕公开什么。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：关系债表。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
2. 定压迫源
   - 判断：家族、公司、婚姻、身份、舆论、法律、亲缘、阶层。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：压力系统。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
3. 分配资源
   - 判断：谁掌握钱、证据、名分、孩子、合同、职位、病历、监控。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：资源表。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
4. 设计误判链
   - 判断：误判从哪里来，谁强化，谁受益，谁能推翻。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：前10集误判链。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
5. 安排站队变化
   - 判断：旁观者、亲密关系、权力方在何时改变判断。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：站队节点。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
6. 设置反派升级
   - 判断：小反派到中层裁判再到幕后规则掌控者。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：压迫阶梯。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。
7. 校验主角行动
   - 判断：每 1-2 集主角必须做一次主动选择。
   - 动作：把判断落到短剧里的具体事件、压力、证据、关系或钩子。
   - 输出：行动清单。
   - 失败信号：只写结论，没有说明哪一集、哪场、哪句话或哪个人物功能出了问题。

## 诊断表

| 问题 | 表面症状 | 对应修复动作 |
| --- | --- | --- |
| 人物只是标签 | 角色只有身份，没有欲望和资源。 | 补欲望、恐惧、掌握物、可造成的损失。 |
| 反派太薄 | 只会骂人陷害。 | 给反派利益逻辑和能压迫主角的规则。 |
| 主角太被动 | 主角只能等待别人发现真相。 | 给主角可主动使用的证据或策略。 |
| 关系债不具体 | 只说有误会。 | 把误会变成可展示物件：照片、病历、转账、监控。 |
| 站队不变化 | 所有人一直同一态度。 | 每 3-5 集至少让一个功能位改变判断。 |
| 助攻无功能 | 朋友只负责解释。 | 让助攻递线索、挡枪、误导或制造行动机会。 |
| 孩子线悬浮 | 萌宝只负责可爱。 | 孩子必须造成认亲、抚养权或证据危机。 |
| 权力系统太小 | 反派失败后没有更大压力。 | 设置更高裁判或规则层。 |

逐项诊断写法：
### 诊断 1：人物只是标签

- 诊断信号：角色只有身份，没有欲望和资源。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：补欲望、恐惧、掌握物、可造成的损失。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 2：反派太薄

- 诊断信号：只会骂人陷害。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：给反派利益逻辑和能压迫主角的规则。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 3：主角太被动

- 诊断信号：主角只能等待别人发现真相。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：给主角可主动使用的证据或策略。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 4：关系债不具体

- 诊断信号：只说有误会。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：把误会变成可展示物件：照片、病历、转账、监控。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 5：站队不变化

- 诊断信号：所有人一直同一态度。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：每 3-5 集至少让一个功能位改变判断。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 6：助攻无功能

- 诊断信号：朋友只负责解释。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：让助攻递线索、挡枪、误导或制造行动机会。
- 输出要求：指出原材料中 1-3 个具体位置，再给出可直接替换的结构或场景写法。
- 自检问题：修复后观众是否能更快理解压力、更想等下一集、且不需要旁白解释？

### 诊断 7：孩子线悬浮

- 诊断信号：萌宝只负责可爱。
- 深层原因：短剧的压力、行动、证据、关系或钩子没有承担应有功能。
- 修复动作：孩子必须造成认亲、抚养权或证据危机。

## 常见失败

| 失败类型 | 后果 | 修正 |
| --- | --- | --- |
| 只讲理论 | 用户拿不到可写内容。 | 每段理论后补一个结构动作或输出字段。 |
| 直接写正文 | 还没确定类型和第一屏。 | 先做任务判断和类型承诺。 |
| 案例复刻 | 新稿像某一部剧。 | 至少混合三个机制，并列必须替换项。 |
| 压力不可见 | 观众不知道主角为何必须行动。 | 改成公开场、可见损失、旁观者站队。 |
| 反击工具缺失 | 主角只靠忍耐和解释。 | 补证据、身份、资源、技能、孩子线索或规则漏洞。 |
| 尾钩模糊 | 每集结尾没有下一集问题。 | 用证据、来人、电话、身份露角或关系选择收尾。 |
| 对白说明化 | 人物替作者解释设定。 | 让信息成为争夺物。 |
| 中段重复 | 同类争吵循环。 | 升级压力来源和裁判层级。 |
| 结尾无清算 | 真相公开但情感债没还。 | 补承认、补偿、放权、公开承担。 |
| 格式混乱 | 正文像小说或分镜秒表。 | 统一剧本格式。 |

## 自检清单

- [ ] 是否先判断了用户任务，而不是直接写？
- [ ] 是否明确了主类型和主追更情绪？
- [ ] 第一屏是否有公开压力、可见损失和旁观者站队？
- [ ] 主角是否有可重复使用的反击工具？
- [ ] 反派或压迫者是否掌握具体资源、规则或证据？
- [ ] 每一集是否有事件，而不是只有情绪对话？
- [ ] 每一集结尾是否留下具体下一集问题？
- [ ] 前3集是否完成压迫、误判和小反击？
- [ ] 前10集是否打开中段更大压力？
- [ ] 60-100集是否有分段追更点？
- [ ] 人物关系是否有可拍的关系债？
- [ ] 对白是否有目标、地位差、信息差和转折？
- [ ] 网文改编是否删除了慢铺垫和无效支线？
- [ ] 对标案例是否只借机制，不借原文表达？
- [ ] 是否列出必须替换的人物、证据、职业、场景和反转顺序？
- [ ] 正文是否使用第几集/场次/场景/人物/动作/对白格式？
- [ ] 质量自检是否指出了仍需补强的地方？
