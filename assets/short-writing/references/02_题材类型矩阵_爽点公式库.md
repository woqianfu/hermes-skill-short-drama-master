## 类型公式与爽点模板

## 2. 爽点阶梯

```text
1级：口头反击。主角让羞辱者短暂失语。
2级：证据反击。主角拿出物件、录音、合同、鉴定、旧账。
3级：资源反击。主角调动外部权力，改留存场站队。
4级：身份反击。主角隐藏身份被部分确认，反派开始恐惧。
5级：公开审判。核心反派在众人面前付出代价。
```

前十集通常使用1-3级，中段逼近4级，终局使用5级。

## 3. 类型混搭规则

- 主赛道只能有一个，它决定第一屏、前十集和追更点。
- 副元素最多两个，只负责新鲜感和中段变化。
- 主副冲突时听主赛道。例如追妻 + 悬疑，悬疑线不能压过追悔情绪。
- 同一集只兑现一个主爽点，另一个元素做钩子即可。

## 题材类型案例机制矩阵

## 1. 分类原则

这次题材类型优化把“霸总、复仇、追妻”等从泛标签拆成可执行的创作模块。每个题材都必须回答：观众等什么、第一屏怎么开、前10集怎么卡、中段怎么换规则、最容易写烂在哪里。

## 6. 题材选择建议

- 用户要强情绪和女性向商业性：优先霸总豪门、追妻火葬场、复仇重生、虐恋误会。
- 用户要轻甜和高互动：优先甜宠契约、萌宝亲子。
- 用户要男性爽感：优先战神强者、赘婿逆袭、神豪财富、小人物逆袭。
- 用户要家庭共鸣：优先婚姻背叛、亲子认亲、婆媳伦理、年代家庭。
- 用户要新鲜感：用悬疑惊悚或古装民国穿越做副元素，但不要抢走主赛道承诺。

## 题材组合规则

题材组合要分主承诺和副刺激。主承诺决定观众为什么追更，副刺激只负责新鲜感。

| 组合 | 正确用法 | 错误用法 |
| --- | --- | --- |
| 追妻 + 悬疑 | 悬疑服务真相迟来和追悔代价 | 悬疑线压过情感清算 |
| 霸总 + 萌宝 | 孩子推动相遇、认亲和偏爱 | 孩子只卖萌不递线索 |
| 复仇 + 重生 | 每个复仇动作对应旧伤 | 主角全知全能一路碾压 |
| 神豪 + 都市爽文 | 财富反差解决现实危机 | 只堆金额没有关系伤害 |
| 家庭伦理 + 亲子 | 血缘秘密改变全家站队 | 只写哭诉没有事件 |
| 古装 + 复仇 | 制度压迫制造低位和清算 | 设定过重，第一屏看不懂 |

## 常见题材错配修正

| 症状 | 原因 | 修正 |
| --- | --- | --- |
| 题材很多但不抓人 | 主赛道不明确 | 删到一个主承诺，其他只做副元素 |
| 霸总不像霸总 | 缺权力差和公开选择 | 补强男主权力、误判代价和偏爱行动 |
| 复仇不爽 | 旧伤不具体 | 给每次清算绑定一个旧伤证据 |
| 追妻不痛 | 男主没有具体伤害 | 让伤害变成可见行动而非误会解释 |
| 神豪无聊 | 只有花钱没有低位委屈 | 先写金钱羞辱，再写资源反差 |
| 家庭伦理重复 | 只吵架不换证据 | 每集改变一个家庭成员站队 |

## Agent 角色设定

类型承诺审片人型 Agent，负责判断短剧到底卖哪一种情绪，并用类型规则约束创作。

工作目标：
- 区分表面题材和真正追更类型。
- 为每个类型列出必备场面、禁区和升级方式。
- 判断混合类型的主副关系，避免爽点互相抵消。
- 为用户创意匹配 3-8 个同类案例机制。
- 把类型承诺转化为前10集的事件密度。

禁止做法：
- 不要用抽象评价代替场景级判断。
- 不要把类型公式当成固定剧情模板。
- 不要复用本地完整剧本的人物、场景、对白或连续桥段。
- 不要在缺少信息时停在追问；先给合理假设并继续输出可执行版本。
- 不要让正文变成小说梗概；正文必须能被演员和导演直接理解。

| 直接写正文 | 还没确定类型和第一屏。 | 先做任务判断和类型承诺。 |
| 案例复刻 | 新稿像某一部剧。 | 至少混合三个机制，并列必须替换项。 |
| 压力不可见 | 观众不知道主角为何必须行动。 | 改成公开场、可见损失、旁观者站队。 |
| 反击工具缺失 | 主角只靠忍耐和解释。 | 补证据、身份、资源、技能、孩子线索或规则漏洞。 |
| 尾钩模糊 | 每集结尾没有下一集问题。 | 用证据、来人、电话、身份露角或关系选择收尾。 |
| 对白说明化 | 人物替作者解释设定。 | 让信息成为争夺物。 |
| 中段重复 | 同类争吵循环。 | 升级压力来源和裁判层级。 |
| 结尾无清算 | 真相公开但情感债没还。 | 补承认、补偿、放权、公开承担。 |
| 格式混乱 | 正文像小说或分镜秒表。 | 统一剧本格式。 |

## 自检清单

- [ ] 是否先判断了用户任务，而不是直接写？
- [ ] 是否明确了主类型和主追更情绪？
- [ ] 第一屏是否有公开压力、可见损失和旁观者站队？
- [ ] 主角是否有可重复使用的反击工具？
- [ ] 反派或压迫者是否掌握具体资源、规则或证据？
- [ ] 每一集是否有事件，而不是只有情绪对话？
- [ ] 每一集结尾是否留下具体下一集问题？
- [ ] 前3集是否完成压迫、误判和小反击？
- [ ] 前10集是否打开中段更大压力？
- [ ] 60-100集是否有分段追更点？
- [ ] 人物关系是否有可拍的关系债？
- [ ] 对白是否有目标、地位差、信息差和转折？
- [ ] 网文改编是否删除了慢铺垫和无效支线？
- [ ] 对标案例是否只借机制，不借原文表达？
- [ ] 是否列出必须替换的人物、证据、职业、场景和反转顺序？
- [ ] 正文是否使用第几集/场次/场景/人物/动作/对白格式？
- [ ] 质量自检是否指出了仍需补强的地方？
